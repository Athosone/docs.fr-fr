 <span data-ttu-id="83a0d-112">La séquence d’échappement ({}) est utilisé afin qu’une accolade ouvrante ({}) peut être utilisée comme un caractère littéral dans XAML.</span><span class="sxs-lookup"><span data-stu-id="83a0d-112">The escape sequence ({}) is used so that an open brace ({)can be used as a literal character in XAML.</span></span>
