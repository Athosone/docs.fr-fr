### <a name="to-create-an-mdi-parent-form-at-design-time"></a><span data-ttu-id="69a5a-109">Pour créer un formulaire MDI parent au moment du design</span><span class="sxs-lookup"><span data-stu-id="69a5a-109">To create an MDI parent form at design time</span></span>

    >  <span data-ttu-id="69a5a-113">Quand vous définissez des propriétés dans la fenêtre **Propriétés**, vous pouvez aussi si vous le souhaitez définir la propriété `WindowState` sur **Maximized**, car il est plus facile de manipuler des fenêtres enfants MDI quand le formulaire parent est agrandi au maximum.</span><span class="sxs-lookup"><span data-stu-id="69a5a-113">While setting properties in the **Properties** window, you can also set the `WindowState` property to **Maximized**, if you like, as it is easiest to manipulate MDI child windows when the parent form is maximized.</span></span> <span data-ttu-id="69a5a-114">Sachez par ailleurs que le contour du formulaire MDI parent prend la couleur système (définie dans le Panneau de configuration Système Windows), et non la couleur d'arrière-plan que vous avez définie avec la propriété <xref:System.Windows.Forms.Control.BackColor%2A?displayProperty=nameWithType>.</span><span class="sxs-lookup"><span data-stu-id="69a5a-114">Additionally, be aware that the edge of the MDI parent form will pick up the system color (set in the Windows System Control Panel), rather than the back color you set using the <xref:System.Windows.Forms.Control.BackColor%2A?displayProperty=nameWithType> property.</span></span>  
  
3. <span data-ttu-id="69a5a-115">Dans la **Boîte à outils**, faites glisser un contrôle **MenuStrip** sur le formulaire.</span><span class="sxs-lookup"><span data-stu-id="69a5a-115">From the **Toolbox**, drag a **MenuStrip** control to the form.</span></span> <span data-ttu-id="69a5a-116">Créez un élément de menu de plus haut niveau en définissant la propriété **Text** sur **&File** avec des éléments de sous-menu appelés **&New** et **&Close**.</span><span class="sxs-lookup"><span data-stu-id="69a5a-116">Create a top-level menu item with the **Text** property set to **&File** with submenu items called **&New** and **&Close**.</span></span> <span data-ttu-id="69a5a-117">Créez également un menu de plus haut niveau appelé **&Window**.</span><span class="sxs-lookup"><span data-stu-id="69a5a-117">Also create a top-level menu item called **&Window**.</span></span>
