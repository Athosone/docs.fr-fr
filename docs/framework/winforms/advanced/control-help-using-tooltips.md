# <a name="control-help-using-tooltips"></a>Affichage sous forme d'info-bulles de l'aide relative aux contrôles
Vous pouvez utiliser le <xref:System.Windows.Forms.ToolTip> composant pour afficher une brève, spécialisé message d’aide pour les contrôles individuels dans les Windows Forms. Le <xref:System.Windows.Forms.ToolTip> composant fournit une propriété qui spécifie le texte affiché pour chaque contrôle sur le formulaire. Pour plus d’informations sur l’utilisation de la <xref:System.Windows.Forms.ToolTip> composant de cette façon, consultez [Comment : Définir des info-bulles pour les contrôles sur un formulaire Windows au moment du Design](../controls/how-to-set-tooltips-for-controls-on-a-windows-form-at-design-time.md). En outre, vous pouvez configurer le <xref:System.Windows.Forms.ToolTip> composant afin qu’il existe un délai avant d’être affiché. Pour plus d'informations, voir [Procédure : Modifier la durée avant affichage du composant ToolTip Windows Forms](../controls/how-to-change-the-delay-of-the-windows-forms-tooltip-component.md).
