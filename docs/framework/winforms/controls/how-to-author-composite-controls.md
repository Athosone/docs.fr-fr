### <a name="to-author-a-composite-control"></a><span data-ttu-id="6260e-111">Pour créer un contrôle composite</span><span class="sxs-lookup"><span data-stu-id="6260e-111">To author a composite control</span></span>  
  
1. <span data-ttu-id="6260e-112">Ouvrez un nouveau projet d’**application Windows** nommé `DemoControlHost`.</span><span class="sxs-lookup"><span data-stu-id="6260e-112">Open a new **Windows Application** project called `DemoControlHost`.</span></span>  
  
2. <span data-ttu-id="6260e-113">Dans le menu **Projet** , cliquez sur **Ajouter un contrôle utilisateur**.</span><span class="sxs-lookup"><span data-stu-id="6260e-113">On the **Project** menu, click **Add User Control**.</span></span>  
  
3. <span data-ttu-id="6260e-114">Dans la boîte de dialogue **Ajouter un nouvel élément**, donnez au fichier de classe (fichier .vb ou .cs) le nom que vous souhaitez pour le contrôle composite.</span><span class="sxs-lookup"><span data-stu-id="6260e-114">In the **Add New Item** dialog box, give the class file (.vb or .cs file) the name you want the composite control to have.</span></span>

6. <span data-ttu-id="6260e-117">Placez le code dans des procédures événementielles afin de gérer les événements déclenchés par le contrôle composite ou par ses contrôles constitutifs.</span><span class="sxs-lookup"><span data-stu-id="6260e-117">Place code in event procedures, to handle events raised by the composite control or by its constituent controls.</span></span>  
  
7. <span data-ttu-id="6260e-118">Fermez le concepteur du contrôle composite et enregistrez le fichier lorsque vous y êtes invité.</span><span class="sxs-lookup"><span data-stu-id="6260e-118">Close the designer for the composite control, and save the file when you are prompted.</span></span>  
  
8. <span data-ttu-id="6260e-119">Dans le menu **Générer** , cliquez sur **Générer la solution**.</span><span class="sxs-lookup"><span data-stu-id="6260e-119">On the **Build** menu, click **Build Solution**.</span></span>  
  
     <span data-ttu-id="6260e-120">Le projet doit être généré de sorte que les contrôles personnalisés apparaissent dans la **boîte à outils**.</span><span class="sxs-lookup"><span data-stu-id="6260e-120">The project must be built in order for custom controls to appear in the **Toolbox**.</span></span>  
  
9. <span data-ttu-id="6260e-121">Utilisez l’onglet **DemoControlHost** de la **boîte à outils** pour ajouter des instances de votre contrôle sur `Form1`.</span><span class="sxs-lookup"><span data-stu-id="6260e-121">Use the **DemoControlHost** tab of the **Toolbox** to add instances of your control to `Form1`.</span></span>  
  
### <a name="to-author-a-control-class-library"></a><span data-ttu-id="6260e-122">Pour créer une bibliothèque de classes de contrôle</span><span class="sxs-lookup"><span data-stu-id="6260e-122">To author a control class library</span></span>  
  
1. <span data-ttu-id="6260e-123">Ouvrez un nouveau projet de **bibliothèque de contrôles Windows**.</span><span class="sxs-lookup"><span data-stu-id="6260e-123">Open a new **Windows Control Library** project.</span></span>  
  
     <span data-ttu-id="6260e-124">Par défaut, le projet contient un contrôle composite.</span><span class="sxs-lookup"><span data-stu-id="6260e-124">By default, the project contains a composite control.</span></span>

3. <span data-ttu-id="6260e-126">Sélectionnez un contrôle dont vous ne souhaitez pas modifier les classes qui héritent, puis définissez la propriété **Modifiers** de ce contrôle sur **Private**.</span><span class="sxs-lookup"><span data-stu-id="6260e-126">Choose a control you do not want inheriting classes to change, and set the **Modifiers** property of this control to **Private**.</span></span>  
  
4. <span data-ttu-id="6260e-127">Créez la DLL.</span><span class="sxs-lookup"><span data-stu-id="6260e-127">Build the DLL.</span></span>  
  
### <a name="to-inherit-from-a-composite-control-in-a-control-class-library"></a><span data-ttu-id="6260e-128">Pour hériter d’un contrôle composite dans une bibliothèque de classes de contrôle</span><span class="sxs-lookup"><span data-stu-id="6260e-128">To inherit from a composite control in a control class library</span></span>  
  
1. <span data-ttu-id="6260e-129">Dans le menu **Fichier**, pointez sur **Ajouter** et sélectionnez **Nouveau projet** pour ajouter un nouveau projet d’**application Windows** à la solution.</span><span class="sxs-lookup"><span data-stu-id="6260e-129">On the **File** menu, point to **Add** and select **New Project** to add a new **Windows Application** project to the solution.</span></span>  
  
2. <span data-ttu-id="6260e-130">Dans l’**Explorateur de solutions**, cliquez avec le bouton droit sur le dossier **References** du nouveau projet, puis sélectionnez **Ajouter une référence** pour ouvrir la boîte de dialogue **Ajouter une référence**.</span><span class="sxs-lookup"><span data-stu-id="6260e-130">In **Solution Explorer**, right-click the **References** folder for the new project and choose **Add Reference** to open the **Add Reference** dialog box.</span></span>  
  
3. <span data-ttu-id="6260e-131">Sélectionnez l’onglet **Projets** et double-cliquez sur votre projet de bibliothèque de contrôles.</span><span class="sxs-lookup"><span data-stu-id="6260e-131">Select the **Projects** tab and double-click your control library project.</span></span>  
  
4. <span data-ttu-id="6260e-132">Dans le menu **Générer** , cliquez sur **Générer la solution**.</span><span class="sxs-lookup"><span data-stu-id="6260e-132">On the **Build** menu, click **Build Solution**.</span></span>  
  
5. <span data-ttu-id="6260e-133">Dans l’**Explorateur de solutions**, cliquez avec le bouton droit sur votre projet de bibliothèque de contrôles, puis sélectionnez **Ajouter un nouvel élément** dans le menu contextuel.</span><span class="sxs-lookup"><span data-stu-id="6260e-133">In **Solution Explorer**, right-click your control library project and select **Add New Item** from the shortcut menu.</span></span>  
  
6. <span data-ttu-id="6260e-134">Sélectionnez le modèle **Contrôle utilisateur hérité** dans la boîte de dialogue **Ajouter un nouvel élément**.</span><span class="sxs-lookup"><span data-stu-id="6260e-134">Select the **Inherited User Control** template from the **Add New Item** dialog box.</span></span>  
  
7. <span data-ttu-id="6260e-135">Dans la boîte de dialogue **Sélecteur d’héritage**, double-cliquez sur le contrôle dont vous voulez hériter.</span><span class="sxs-lookup"><span data-stu-id="6260e-135">In the **Inheritance Picker** dialog box, double-click the control you want to inherit from.</span></span>  
  
     <span data-ttu-id="6260e-136">Un nouveau contrôle est ajouté à votre projet.</span><span class="sxs-lookup"><span data-stu-id="6260e-136">A new control is added to your project.</span></span>  
  
8. <span data-ttu-id="6260e-137">Ouvrez le concepteur visuel du nouveau contrôle et ajoutez d’autres contrôles constitutifs.</span><span class="sxs-lookup"><span data-stu-id="6260e-137">Open the visual designer for the new control and add additional constituent controls.</span></span>  
  
     <span data-ttu-id="6260e-138">Vous pouvez voir les contrôles constitutifs hérités du contrôle composite dans votre DLL, et vous pouvez modifier les propriétés des contrôles dont la propriété **Modifiers** est définie sur **Public**.</span><span class="sxs-lookup"><span data-stu-id="6260e-138">You can see the constituent controls that were inherited from the composite control in your DLL, and you can alter the properties of controls whose **Modifiers** property is **Public**.</span></span> <span data-ttu-id="6260e-139">Vous ne pouvez pas modifier les propriétés du contrôle dont la propriété **Modifiers** est définie sur **Private**.</span><span class="sxs-lookup"><span data-stu-id="6260e-139">You cannot change the properties of the control whose **Modifiers** property is **Private**.</span></span>
