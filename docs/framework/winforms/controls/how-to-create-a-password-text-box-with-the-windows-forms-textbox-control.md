# <a name="how-to-create-a-password-text-box-with-the-windows-forms-textbox-control"></a><span data-ttu-id="3f2d6-102">Procédure : créer une zone de texte pour un mot de passe avec le contrôle TextBox Windows Forms</span><span class="sxs-lookup"><span data-stu-id="3f2d6-102">How to: Create a Password Text Box with the Windows Forms TextBox Control</span></span>
<span data-ttu-id="3f2d6-103">Une zone de mot de passe est une zone de texte Windows Forms qui affiche les caractères d’espace réservé pendant qu’un utilisateur tape une chaîne.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-103">A password box is a Windows Forms text box that displays placeholder characters while a user types a string.</span></span>  
  
### <a name="to-create-a-password-text-box"></a><span data-ttu-id="3f2d6-104">Pour créer une zone de texte mot de passe</span><span class="sxs-lookup"><span data-stu-id="3f2d6-104">To create a password text box</span></span>  
  
1. <span data-ttu-id="3f2d6-105">Définir le <xref:System.Windows.Forms.TextBox.PasswordChar%2A> propriété de la <xref:System.Windows.Forms.TextBox> contrôle à un caractère spécifique.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-105">Set the <xref:System.Windows.Forms.TextBox.PasswordChar%2A> property of the <xref:System.Windows.Forms.TextBox> control to a specific character.</span></span>  
  
     <span data-ttu-id="3f2d6-106">Le <xref:System.Windows.Forms.TextBox.PasswordChar%2A> propriété spécifie le caractère affiché dans la zone de texte.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-106">The <xref:System.Windows.Forms.TextBox.PasswordChar%2A> property specifies the character displayed in the text box.</span></span> <span data-ttu-id="3f2d6-107">Par exemple, si vous souhaitez que des astérisques s’affichent dans la zone de mot de passe, spécifiez \* pour le <xref:System.Windows.Forms.TextBox.PasswordChar%2A> propriété dans la fenêtre Propriétés.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-107">For example, if you want asterisks displayed in the password box, specify \* for the <xref:System.Windows.Forms.TextBox.PasswordChar%2A> property in the Properties window.</span></span> <span data-ttu-id="3f2d6-108">Ensuite, indépendamment du caractère qu’un utilisateur tape dans la zone de texte, un astérisque est affiché.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-108">Then, regardless of what character a user types in the text box, an asterisk is displayed.</span></span>  
  
2. <span data-ttu-id="3f2d6-109">(Facultatif) Définir le <xref:System.Windows.Forms.TextBoxBase.MaxLength%2A> propriété.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-109">(Optional) Set the <xref:System.Windows.Forms.TextBoxBase.MaxLength%2A> property.</span></span> <span data-ttu-id="3f2d6-110">La propriété détermine le nombre de caractères peut être tapé dans la zone de texte.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-110">The property determines how many characters can be typed in the text box.</span></span> <span data-ttu-id="3f2d6-111">Si la longueur maximale est dépassée, le système émet un signal sonore et la zone de texte n’accepte pas plus de caractères.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-111">If the maximum length is exceeded, the system emits a beep and the text box does not accept any more characters.</span></span> <span data-ttu-id="3f2d6-112">Notez que vous ne pouvez pas effectuer cette opération car la longueur maximale d’un mot de passe peut-être être utiles aux pirates informatiques qui tentent de deviner le mot de passe.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-112">Note that you may not wish to do this as the maximum length of a password may be of use to hackers who are trying to guess the password.</span></span>  
  
     <span data-ttu-id="3f2d6-113">L’exemple de code suivant montre comment initialiser une zone de texte qui acceptent une chaîne de 14 caractères au maximum et affichera des astérisques à la place de la chaîne.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-113">The following code example shows how to initialize a text box that will accept a string up to 14 characters long and display asterisks in place of the string.</span></span> <span data-ttu-id="3f2d6-114">Le `InitializeMyControl` procédure s’exécutera pas automatiquement ; il doit être appelé.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-114">The `InitializeMyControl` procedure will not execute automatically; it must be called.</span></span>  
  
    > [!IMPORTANT]
    >  <span data-ttu-id="3f2d6-115">À l’aide de la <xref:System.Windows.Forms.TextBox.PasswordChar%2A> propriété sur une zone de texte permet de garantir que les autres personnes ne sera pas en mesure de déterminer un mot de passe utilisateur observant l’entrer.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-115">Using the <xref:System.Windows.Forms.TextBox.PasswordChar%2A> property on a text box can help ensure that other people will not be able to determine a user's password if they observe the user entering it.</span></span> <span data-ttu-id="3f2d6-116">Cette mesure de sécurité ne couvre pas une forme quelconque de stockage ou transmission du mot de passe qui peut se produire en raison de votre logique d’application.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-116">This security measure does not cover any sort of storage or transmission of the password that can occur due to your application logic.</span></span> <span data-ttu-id="3f2d6-117">Étant donné que le texte entré n’est pas chiffré en aucune façon, vous devez le considérer comme vous le feriez pour d’autres données confidentielles.</span><span class="sxs-lookup"><span data-stu-id="3f2d6-117">Because the text entered is not encrypted in any way, you should treat it as you would any other confidential data.</span></span> <span data-ttu-id="3f2d6-118">Même si elle n’apparaît pas en tant que tel, le mot de passe est toujours traité comme une chaîne de texte brut (sauf si vous avez implémenté certaines mesures de sécurité supplémentaires).</span><span class="sxs-lookup"><span data-stu-id="3f2d6-118">Even though it does not appear as such, the password is still being treated as a plain-text string (unless you have implemented some additional security measure).</span></span>  
  
    ```vb  
    Private Sub InitializeMyControl()  
       ' Set to no text.  
       TextBox1.Text = ""  
       ' The password character is an asterisk.  
       TextBox1.PasswordChar = "*"  
       ' The control will allow no more than 14 characters.  
       TextBox1.MaxLength = 14  
    End Sub  
    ```  
  
    ```csharp  
    private void InitializeMyControl()  
    {  
       // Set to no text.  
       textBox1.Text = "";  
       // The password character is an asterisk.  
       textBox1.PasswordChar = '*';  
       // The control will allow no more than 14 characters.  
       textBox1.MaxLength = 14;  
    }  
    ```  
  
    ```cpp  
    private:  
       void InitializeMyControl()  
       {  
          // Set to no text.  
          textBox1->Text = "";  
          // The password character is an asterisk.  
          textBox1->PasswordChar = '*';  
          // The control will allow no more than 14 characters.  
          textBox1->MaxLength = 14;  
       }  
    ```
