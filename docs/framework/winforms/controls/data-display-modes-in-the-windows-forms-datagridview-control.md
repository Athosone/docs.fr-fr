## <a name="virtual"></a><span data-ttu-id="df94c-129">Virtuel</span><span class="sxs-lookup"><span data-stu-id="df94c-129">Virtual</span></span>  
 <span data-ttu-id="df94c-130">Avec le mode virtuel, vous pouvez implémenter vos propres opérations de gestion de données.</span><span class="sxs-lookup"><span data-stu-id="df94c-130">With virtual mode, you can implement your own data management operations.</span></span> <span data-ttu-id="df94c-131">Cela est nécessaire pour conserver les valeurs des colonnes indépendantes en mode dépendant lorsque le contrôle est trié par colonnes dépendantes.</span><span class="sxs-lookup"><span data-stu-id="df94c-131">This is necessary to maintain the values of unbound columns in bound mode when the control is sorted by bound columns.</span></span> <span data-ttu-id="df94c-132">La principale utilisation du mode virtuel est toutefois, pour optimiser les performances lors de l’interaction avec grandes quantités de données.</span><span class="sxs-lookup"><span data-stu-id="df94c-132">The primary use of virtual mode, however, is to optimize performance when interacting with large amounts of data.</span></span>  
  
 <span data-ttu-id="df94c-133">Vous attachez le <xref:System.Windows.Forms.DataGridView> contrôle à un cache que vous gérez et votre code contrôle quand les lignes de données sont envoyées et extraites.</span><span class="sxs-lookup"><span data-stu-id="df94c-133">You attach the <xref:System.Windows.Forms.DataGridView> control to a cache that you manage, and your code controls when data rows are pushed and pulled.</span></span> <span data-ttu-id="df94c-134">Pour minimiser l’encombrement mémoire, le cache doit être similaire au nombre de lignes actuellement affichées.</span><span class="sxs-lookup"><span data-stu-id="df94c-134">To keep the memory footprint small, the cache should be similar in size to the number of rows currently displayed.</span></span> <span data-ttu-id="df94c-135">Lorsque l’utilisateur fait défiler de nouvelles lignes dans la vue, votre code demande de nouvelles données à partir du cache et vide éventuellement les anciennes données de la mémoire.</span><span class="sxs-lookup"><span data-stu-id="df94c-135">When the user scrolls new rows into view, your code requests new data from the cache and optionally flushes old data from memory.</span></span>
