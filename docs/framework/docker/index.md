# <a name="deploying-net-framework-applications-with-docker"></a>Déploiement d’applications .NET Framework avec Docker

Vous pouvez déployer une application .NET Framework avec Docker à l’aide de conteneurs Windows. Découvrez les conditions d’utilisation des [conteneurs Windows](/virtualization/windowscontainers/about/) et comment [bien démarrer avec Docker pour Windows](https://docs.docker.com/docker-for-windows/). 

Vous pouvez commencer par [exécuter une application console avec Docker](console.md).
Pour exécuter des applications web dans Docker, vous pouvez lire [Applications ASP.NET MVC dans Docker](/aspnet/mvc/overview/deployment/docker-aspnetmvc).
