# <a name="transport-extensibility"></a>Extensibilité du transport
Cette section contient des exemples qui illustrent les transports personnalisés dans Windows Communication Foundation (WCF).  
  
## <a name="in-this-section"></a>Dans cette section  
 [Activation UDP](../../../../docs/framework/wcf/samples/udp-activation.md)  
 Montre comment étendre le [Transport : UDP](../../../../docs/framework/wcf/samples/transport-udp.md) exemple pour prendre en charge l’activation de processus en utilisant le Service de l’Activation des processus Windows (WAS).  
  
 [Transport : Transactions personnalisées sur UDP](../../../../docs/framework/wcf/samples/transport-custom-transactions-over-udp-sample.md)  
 Montre comment étendre l'exemple UDP Transport afin de prendre en charge le flux de transactions personnalisé et présente l'utilisation de la propriété <xref:System.ServiceModel.Channels.TransactionMessageProperty>.  
  
 [Transport : Interopérabilité TCP WSE 3.0](../../../../docs/framework/wcf/samples/transport-wse-3-0-tcp-interoperability.md)  
 Cet exemple montre comment implémenter une session duplex TCP comme transport personnalisé Windows Communication Foundation (WCF).
