# <a name="protocolstopfailure"></a>ProtocolStopFailure
Id : 147  
  
 Gravité : Error  
  
 Catégorie : TransactionBridge  
  
## <a name="description"></a>Description  
 Cet événement indique que le service de protocole WS-Atomic Transaction n'a pas pu être arrêté. L'événement répertorie l'ID de protocole, le nom de protocole, l'exception, le nom de processus et l'ID de processus.
