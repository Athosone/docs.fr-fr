# <a name="transportauthenticationfailure"></a>TransportAuthenticationFailure
Id : 174  
  
 Gravité : Error  
  
 Catégorie : SecurityAudit  
  
## <a name="description"></a>Description  
 Cet événement indique que l'authentification de transport a échoué, en répertoriant l'identité du client et l'ID d'activité.
