# <a name="messagequeueduplicatedpipeleak"></a>MessageQueueDuplicatedPipeLeak
Id : 166  
  
 Gravité : Error  
  
 Catégorie : SMSvcHost  
  
## <a name="description"></a>Description  
 Cet événement indique que l'erreur s'est produite lors de la distribution du canal nommé dupliqué. Ce descripteur est maintenant intégré dans le processus. Cet événement répertorie la source, l'exception, l'ID et le nom du processus.
