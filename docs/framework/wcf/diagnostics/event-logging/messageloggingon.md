# <a name="messageloggingon"></a>MessageLoggingOn
Id : 116  
  
 Gravité : Error  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 La journalisation des messages a été activée. Les informations sensibles peuvent être enregistrées, même si elles sont chiffrées sur la connexion. Par exemple, les corps de message. L'événement répertorie l'AppID, l'exception, le nom de processus et l'ID de processus.
