# <a name="servicestartfailed"></a>ServiceStartFailed
Id : 164  
  
 Gravité : Error  
  
 Catégorie : SMSvcHost  
  
## <a name="description"></a>Description  
 Cet événement indique qu'une demande pour démarrer le service a échoué. Il répertorie le code d'erreur, le nom de processus et l'ID de processus.
