# <a name="wmiregistrationfailed"></a>WmiRegistrationFailed
Id : 126  
  
 Gravité : Error  
  
 Catégorie : ServiceModel  
  
## <a name="description"></a>Description  
 Cet événement indique que le fournisseur WMI n'a pas été enregistré. L'événement répertorie l'objet WMI, l'erreur, le nom de processus et l'ID de processus.
