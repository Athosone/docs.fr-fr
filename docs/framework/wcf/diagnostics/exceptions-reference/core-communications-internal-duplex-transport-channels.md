# <a name="core-communications-internal-duplex-transport-channels"></a>Communications principales : Canaux de transport duplex internes
Cette rubrique répertorie toutes les exceptions générées par les canaux de Transport Duplex internes Windows Communication Foundation (WCF).  
  
## <a name="exception-list"></a>Liste des exceptions  
  
|Code de la ressource|Chaîne de la ressource|  
|-------------------|---------------------|  
|MessageMustHaveViaOrToSetForSendingOnServerSideCompositeDuplexChannels|Pour envoyer un message sur des canaux duplex composites serveur, la propriété Via ou l'en-tête To doit être défini dans le message.|
