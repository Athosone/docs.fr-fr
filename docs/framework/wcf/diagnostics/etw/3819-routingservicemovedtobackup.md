## <a name="description"></a>Description  
 Cet événement est émis lorsque le service de routage a été déplacé vers les points de terminaison de sauvegarde.  
  
## <a name="message"></a>Message  
 Message du service de routage avec l'ID %1 [opération %2] déplacé vers le point de terminaison de sauvegarde « %3 ».
