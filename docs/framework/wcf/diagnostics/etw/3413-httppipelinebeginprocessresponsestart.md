# <a name="3413---httppipelinebeginprocessresponsestart"></a>3413 - HttpPipelineBeginProcessResponseStart
## <a name="properties"></a>Properties  
  
|||  
|-|-|  
|Id|3413|  
|Mots clés|HTTP|  
|Niveau|Verbose|  
|Canal|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  
 Cet événement est émis lorsque le gestionnaire de messages HTTP a commencé le traitement de la réponse de façon asynchrone.  
  
## <a name="message"></a>Message  
 Le gestionnaire de messages HTTP a démarré le traitement asynchrone de la réponse.
