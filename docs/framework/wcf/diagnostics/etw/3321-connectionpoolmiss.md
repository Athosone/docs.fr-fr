## <a name="description"></a>Description  
 Cet événement est émis lorsque le pool n'a aucune connexion disponible.  
  
## <a name="message"></a>Message  
 Le pool de %1 ne dispose d'aucune connexion disponible et dispose de %2 connexions occupées.
