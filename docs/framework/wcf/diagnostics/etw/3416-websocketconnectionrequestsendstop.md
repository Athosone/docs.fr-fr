## <a name="properties"></a>Properties  
  
|||  
|-|-|  
|Id|3416|  
|Mots clés|HTTP|  
|Niveau|Verbose|  
|Canal|Microsoft-Windows-Application Server-Applications/Débogage|  
  
## <a name="description"></a>Description  
 Cet événement est émis lorsque l'envoi d'une requête de connexion WebSocket s'est arrêté.  
  
## <a name="message"></a>Message  
 Demande de connexion à l'élément WebSocketId %1 envoyée.
