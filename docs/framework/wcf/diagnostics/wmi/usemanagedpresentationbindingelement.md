# <a name="usemanagedpresentationbindingelement"></a>UseManagedPresentationBindingElement
UseManagedPresentationBindingElement  
  
## <a name="syntax"></a>Syntaxe  
  
```csharp
class UseManagedPresentationBindingElement : BindingElement  
{  
};  
```  
  
## <a name="methods"></a>Méthodes  
 La classe UseManagedPresentationBindingElement hérite de BindingElement mais ne définit pas de méthodes ou de propriétés supplémentaires.

|MOF|Déclaré dans Servicemodel.mof.|  
|---------|-----------------------------------|  
|Espace de noms|Défini dans root\ServiceModel|
