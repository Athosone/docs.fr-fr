# <a name="getoperationcounterinstancename"></a>GetOperationCounterInstanceName
Récupère le nom d'instance du compteur de performance d'opération.  
  
## <a name="syntax"></a>Syntaxe  
  
```csharp
string GetOperationCounterInstanceName(  
  string Operation  
);  
```  
  
## <a name="parameters"></a>Paramètres  
  
### <a name="operation"></a>Opération  
 Chaîne [in]  
  
## <a name="requirements"></a>Configuration requise  
  
|MOF|Déclaré dans Servicemodel.mof.|  
|---------|-----------------------------------|  
|Espace de noms|Défini dans root\ServiceModel|
