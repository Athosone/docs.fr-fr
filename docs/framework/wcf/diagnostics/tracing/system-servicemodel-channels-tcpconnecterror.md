# <a name="systemservicemodelchannelstcpconnecterror"></a>System.ServiceModel.Channels.TcpConnectError
L'opération de connexion TCP a échoué.  
  
## <a name="description"></a>Description  
 Ce suivi de niveau d'avertissement indique l'échec d'une connexion à un point de terminaison TCP. Cela peut se produire si le point de terminaison distant ne répond pas à un port et une adresse IP donnés. Ce suivi peut être ignoré si les tentatives ultérieures de connexion à d'autres adresses IP valides (telles que des adresses IPv4 ou IPv6 ou d'autres adresses représentant un nom d'hôte donné) aboutissent. L'exception dans ce suivi peut indiquer des informations supplémentaires sur l'erreur.
