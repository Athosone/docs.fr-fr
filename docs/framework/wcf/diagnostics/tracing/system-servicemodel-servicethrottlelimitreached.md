# <a name="systemservicemodelservicethrottlelimitreached"></a>System.ServiceModel.ServiceThrottleLimitReached
System.ServiceModel.ServiceThrottleLimitReached  
  
## <a name="description"></a>Description  
 Une valeur de l'accélérateur MaxConcurrentCalls/Instances/Sessions a été atteinte et un message est mis en file d'attente.  
  
 Vous pouvez utiliser ce suivi pour ajuster les valeurs de l'accélérateur en fonction de la charge de production attendue et réelle.
