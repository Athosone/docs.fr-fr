# <a name="systemservicemodelservicehosttimeoutonclose"></a>System.ServiceModel.ServiceHostTimeoutOnClose
System.ServiceModel.ServiceHostTimeoutOnClose  
  
## <a name="description"></a>Description  
 L'opération de fermeture d'un ServiceHost a dépassé le délai d'attente. Cela indique que certains canaux de session n'ont pas été arrêtés correctement par le client et ont pu être abandonnés. Modifiez la durée par défaut afin d'ajuster le délai d'attente de fermeture.
