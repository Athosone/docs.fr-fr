# <a name="debugging-on-the-client"></a>Débogage sur le client
Pour faciliter aux utilisateurs d’écrire des applications clientes pour votre service WCF, vous pouvez ajouter la [ \<serviceDebug >](../../../../../docs/framework/configure-apps/file-schema/wcf/servicedebug.md) comportement au fichier de configuration de votre service de service. Ce comportement permet de publier des pages d'aide et de retourner des informations d'exception gérées dans les détails des erreurs SOAP renvoyées au client.
