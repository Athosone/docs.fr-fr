# <a name="systemservicemodelchannelsprematuredatagrameof"></a>System.ServiceModel.Channels.PrematureDatagramEof
System.ServiceModel.Channels.PrematureDatagramEof  
  
## <a name="description"></a>Description  
 Un message Null (extrémité de signalisation d'un canal) a été reçu d'un canal de datagramme, mais le canal est toujours dans l'état Opened. Cela indique une erreur dans le canal de datagramme, et la boucle de réception de démultiplexeur a été prématurément arrêtée.
