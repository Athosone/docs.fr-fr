# <a name="queued-poison-messages"></a>Messages incohérents mis en file d'attente
Nom du compteur : Messages incohérents en file d’attente.  
  
## <a name="description"></a>Description  
 Nombre de messages vers ce service marqués comme étant incohérents par le transport de mise en file d'attente.  
  
 Consultez [des messages incohérents](https://go.microsoft.com/fwlink/?LinkID=96233) pour plus d’informations sur quand les messages sont marqués comme incohérents.
