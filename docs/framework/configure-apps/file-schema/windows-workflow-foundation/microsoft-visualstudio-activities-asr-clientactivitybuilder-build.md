# <a name="microsoftvisualstudioactivitiesasrclientactivitybuilderbuild"></a>Microsoft.VisualStudio.Activities.Asr.ClientActivityBuilder.Build
Crée et configure un objet <xref:System.Activities.ActivityBuilder> qui fournit des données pour une activité de workflow.  
  
## <a name="syntax"></a>Syntaxe  
  
```csharp  
public ActivityBuilder Build();   
```  
  
## <a name="return-value"></a>Valeur de retour  
 Retourne un objet <xref:System.Activities.ActivityBuilder>.
