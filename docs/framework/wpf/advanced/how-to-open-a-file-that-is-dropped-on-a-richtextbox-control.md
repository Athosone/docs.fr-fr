<span data-ttu-id="bfa7c-103">Dans [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)], le <xref:System.Windows.Controls.TextBox>, <xref:System.Windows.Controls.RichTextBox>, et <xref:System.Windows.Documents.FlowDocument> tous les contrôles possèdent des fonctionnalités de glisser-déplacer intégrées.</span><span class="sxs-lookup"><span data-stu-id="bfa7c-103">In [!INCLUDE[TLA#tla_winclient](../../../../includes/tlasharptla-winclient-md.md)], the <xref:System.Windows.Controls.TextBox>, <xref:System.Windows.Controls.RichTextBox>, and <xref:System.Windows.Documents.FlowDocument> controls all have built-in drag-and-drop functionality.</span></span> <span data-ttu-id="bfa7c-104">La fonctionnalité intégrée permet de glisser-déplacer du texte au sein et entre les contrôles.</span><span class="sxs-lookup"><span data-stu-id="bfa7c-104">The built-in functionality enables drag-and-drop of text within and between the controls.</span></span> <span data-ttu-id="bfa7c-105">Toutefois, elles n’autorisent pas l’ouverture d’un fichier en supprimant le fichier sur le contrôle.</span><span class="sxs-lookup"><span data-stu-id="bfa7c-105">However, it does not enable opening a file by dropping the file on the control.</span></span> <span data-ttu-id="bfa7c-106">Ces contrôles marquent également les événements de glisser-déplacer comme étant géré.</span><span class="sxs-lookup"><span data-stu-id="bfa7c-106">These controls also mark the drag-and-drop events as handled.</span></span> <span data-ttu-id="bfa7c-107">Par conséquent, par défaut, vous ne pouvez pas ajouter vos propres gestionnaires d’événements pour fournir des fonctionnalités pour ouvrir des fichiers effacés.</span><span class="sxs-lookup"><span data-stu-id="bfa7c-107">As a result, by default, you cannot add your own event handlers to provide functionality to open dropped files.</span></span>  
  
 <span data-ttu-id="bfa7c-108">Pour ajouter un traitement supplémentaire pour les événements de glisser-déplacer dans ces contrôles, utilisez le <xref:System.Windows.UIElement.AddHandler%28System.Windows.RoutedEvent%2CSystem.Delegate%2CSystem.Boolean%29> méthode pour ajouter vos gestionnaires d’événements pour les événements de glisser-déplacer.</span><span class="sxs-lookup"><span data-stu-id="bfa7c-108">To add additional handling for drag-and-drop events in these controls, use the <xref:System.Windows.UIElement.AddHandler%28System.Windows.RoutedEvent%2CSystem.Delegate%2CSystem.Boolean%29> method to add your event handlers for the drag-and-drop events.</span></span> <span data-ttu-id="bfa7c-109">Définir le `handledEventsToo` paramètre `true` pour que le gestionnaire spécifié soit appelé pour un événement routé qui a déjà été marqué comme géré par un autre élément le long de l’itinéraire d’événement.</span><span class="sxs-lookup"><span data-stu-id="bfa7c-109">Set the `handledEventsToo` parameter to `true` to have the specified handler be invoked for a routed event that has already been marked as handled by another element along the event route.</span></span>  
  
> [!TIP]
>  <span data-ttu-id="bfa7c-110">Vous pouvez remplacer la fonctionnalité de glisser-déplacer intégrée de <xref:System.Windows.Controls.TextBox>, <xref:System.Windows.Controls.RichTextBox>, et <xref:System.Windows.Documents.FlowDocument> en gérant les versions préliminaires des événements de glisser-déplacer et de marquer les événements d’aperçu comme géré.</span><span class="sxs-lookup"><span data-stu-id="bfa7c-110">You can replace the built-in drag-and-drop functionality of <xref:System.Windows.Controls.TextBox>, <xref:System.Windows.Controls.RichTextBox>, and <xref:System.Windows.Documents.FlowDocument> by handling the preview versions of the drag-and-drop events and marking the preview events as handled.</span></span> <span data-ttu-id="bfa7c-111">Toutefois, cela va désactiver la fonctionnalité de glisser-déplacer intégrée et n’est pas recommandée.</span><span class="sxs-lookup"><span data-stu-id="bfa7c-111">However, this will disable the built-in drag-and-drop functionality, and is not recommended.</span></span>
