<span data-ttu-id="c127d-103">Le processus d'inférence d'un schéma de l'objet <xref:System.Data.DataSet> à partir de XML peut aboutir à des schémas différents en fonction des éléments XML figurant dans chaque document.</span><span class="sxs-lookup"><span data-stu-id="c127d-103">The process of inferring a <xref:System.Data.DataSet> schema from XML can result in different schemas depending on the XML elements in each document.</span></span> <span data-ttu-id="c127d-104">Examinons, par exemple, les documents XML suivants.</span><span class="sxs-lookup"><span data-stu-id="c127d-104">For example, consider the following XML documents.</span></span>

 <span data-ttu-id="c127d-106">Document2 :</span><span class="sxs-lookup"><span data-stu-id="c127d-106">Document2:</span></span>
