# <a name="subscript-out-of-range-visual-basic"></a>Indice hors limites (Visual Basic)
Un indice de tableau n’est pas valide, car il se situe en dehors de la plage autorisée. La valeur la plus faible indice pour une dimension est toujours 0 et la valeur d’indice le plus élevée est retournée par la `GetUpperBound` méthode pour cette dimension.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Modifiez l’indice de sorte qu’il se situe dans la plage valide.
