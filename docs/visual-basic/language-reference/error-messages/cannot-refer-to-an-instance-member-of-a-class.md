 <span data-ttu-id="d9438-110">Toutefois, la procédure `setX` est `Shared` entre toutes les instances de classe `sample`.</span><span class="sxs-lookup"><span data-stu-id="d9438-110">However, the procedure `setX` is `Shared` among all instances of class `sample`.</span></span> <span data-ttu-id="d9438-111">Cela signifie qu’il n’est pas associé à une instance de la classe, mais au lieu de cela fonctionne indépendamment des instances individuelles.</span><span class="sxs-lookup"><span data-stu-id="d9438-111">This means it is not associated with any one instance of the class, but rather operates independently of individual instances.</span></span> <span data-ttu-id="d9438-112">Car il ne possède aucune connexion avec une instance particulière, `setX` ne peut pas accéder à une variable d’instance.</span><span class="sxs-lookup"><span data-stu-id="d9438-112">Because it has no connection with a particular instance, `setX` cannot access an instance variable.</span></span> <span data-ttu-id="d9438-113">Il doit fonctionner uniquement sur `Shared` variables.</span><span class="sxs-lookup"><span data-stu-id="d9438-113">It must operate only on `Shared` variables.</span></span> <span data-ttu-id="d9438-114">Lorsque `setX` définit ou modifie la valeur d’une variable partagée, que la nouvelle valeur est disponible pour toutes les instances de la classe.</span><span class="sxs-lookup"><span data-stu-id="d9438-114">When `setX` sets or changes the value of a shared variable, that new value is available to all instances of the class.</span></span>
