# <a name="path-not-found"></a>Chemin d'accès introuvable
Pendant une opération d’accès au fichier ou d’accès au disque, le système d’exploitation n’a pas pu trouver le chemin d’accès spécifié. Le chemin d’accès à un fichier comprend la spécification de lecteur ainsi que les répertoires et les sous-répertoires qui doivent être parcourus pour localiser le fichier. Un chemin peut être relatif ou absolu.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
-   Vérifiez et spécifiez à nouveau le chemin d’accès.
