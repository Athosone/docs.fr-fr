# <a name="name-cannot-expose-type-type1-in-specifier-type2-typename-through-name"></a>'\<nom >' ne peut pas exposer le type \<type1 > dans \<spécificateur > \<type2 > '\<nom_type >' via \<nom >
Vous avez tenté d’exposer un type privé en dehors de la classe publique `MyClass`.  
  
 **ID d’erreur :** BC30508  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Déclarez le type `Public`.
