# <a name="nullable-modifier-cannot-be-specified-on-both-a-variable-and-its-type"></a>Impossible de spécifier un modificateur autorisant les valeurs Null pour la variable et son type
Le modificateur de type Nullable (?) a été inclus à la fois pour une variable déclarée et son type déclaré.  
  
 **ID d’erreur :** BC33100  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Supprimez le modificateur de type Nullable (?) de la variable ou de son type déclaré.
