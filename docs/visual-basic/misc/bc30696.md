# <a name="membername1-and-membername2-cannot-overload-each-other-because-they-differ-only-by-the-types-of-optional-parameters"></a>«\<nom_membre1 >' et '\<nom_membre2 >' ne peut pas se surcharger mutuellement, car seuls les types des paramètres optionnels les différencient
Les membres surchargés doivent avoir des types de données différents pour un ou plusieurs paramètres obligatoires.  
  
 **ID d’erreur :** BC30696  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Définissez au moins un paramètre obligatoire en tant que type de données différent.
