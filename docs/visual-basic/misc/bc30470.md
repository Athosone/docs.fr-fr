# <a name="myclass-cannot-be-used-outside-of-a-class"></a>'MyClass' ne peut pas être utilisé à l’extérieur d’une classe
Vous avez utilisé le mot clé `MyClass` en dehors d’une structure de classe.  
  
 **ID d’erreur :** BC30470  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Déplacez la déclaration `MyClass` vers votre structure de classe.
