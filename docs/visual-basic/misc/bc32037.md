# <a name="next-statement-names-more-variables-than-there-are-matching-for-statements"></a><span data-ttu-id="0b1a4-102">L’instruction 'Next' nomme plus de variables qu’il n’existe d’instructions 'For' correspondantes</span><span class="sxs-lookup"><span data-stu-id="0b1a4-102">'Next' statement names more variables than there are matching 'For' statements</span></span>
<span data-ttu-id="0b1a4-103">Les boucles imbriquées sont terminées avec une seule instruction `Next` qui spécifie plus de variables de boucle qu’il n’y en a dans l’imbrication, comme dans l’exemple suivant :</span><span class="sxs-lookup"><span data-stu-id="0b1a4-103">Nested loops are terminated with a single `Next` statement that specifies more loop variables than there are loops in the nest, as in the following example:</span></span>  
  
```  
For I = 1 To 10  
   For J = 1 To 5  
      ' ...  
Next J, I, K   ' Next J, I is valid, but there is no loop on K.  
```
