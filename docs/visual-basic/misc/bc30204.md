# <a name="integer-constant-expected"></a>Constante Integer attendue
Il existe une occurrence de directive `#ExternalSource` dans laquelle le deuxième argument n’est pas un littéral d’entier. Seul un littéral d’entier est valide dans ce contexte. Une constante nommée ou un membre d’énumération n’est pas valide.  
  
 **ID d’erreur :** BC30204  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Utilisez une constante nommée ou un membre d’énumération à la place d’un littéral.  
  
2. Fournissez un littéral d’entier comme deuxième argument à la directive `#ExternalSource` .
