# <a name="goto-to-label-labelname-cannot-be-used-within-lambda-expressions"></a>L’étiquette GoTo '\<nom_étiquette >' ne peut pas être utilisé dans les expressions lambda
Le corps d’une expression lambda doit être une expression qui calcule et retourne une valeur unique. Aucune instruction n’est autorisée, y compris les instructions `GoTo` .  
  
 **ID d’erreur :** BC36597
