# <a name="fully-qualified-names-for-types-in-namespaces-cannot-be-longer-than-number-characters"></a>Noms qualifiés complets pour les types dans les espaces de noms ne peut pas dépasser \<nombre > caractères
Le nom qualifié complet du type ne peut pas dépasser le nombre spécifié de caractères.  
  
 **ID d’erreur :** BC30031  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Raccourcissez le nom du type.
