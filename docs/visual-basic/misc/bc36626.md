# <a name="lambda-expression-cannot-be-converted-to-typename-because-type-typename-is-declared-mustinherit-and-cannot-be-created"></a>Expression lambda ne peut pas être convertie en '\<typename >', car type '\<nom_type >' est déclaré 'MustInherit' et ne peut pas être créé
Une instruction tente de convertir une expression lambda en un type qui peut être uniquement une classe de base et ne peut pas être utilisé pour créer une instance.  
  
 **ID d’erreur :** BC36626
