# <a name="method-methodname-cannot-be-declared-partial-because-only-one-method-methodname-can-be-marked-partial"></a>Méthode '\<nom_méthode >' ne peut pas être déclaré 'Partial', car qu’une seule méthode '\<nom_méthode >' peut être marquée comme 'Partial'
Vous ne pouvez avoir deux déclarations de méthode partielle ayant les mêmes nom et signature.  
  
 **ID d’erreur :** BC31433  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Modifiez le nom ou la signature de l’une des déclarations de méthode partielle.
