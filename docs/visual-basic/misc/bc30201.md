# <a name="expression-expected"></a>Expression attendue
Un opérateur tel que `+` ou `=` nécessite une expression pour son opérande de droite, mais aucune expression reconnaissable n’apparaît.  
  
 **ID d’erreur :** BC30201  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Fournissez une expression valide pour l’opérande de droite de l’opérateur.
