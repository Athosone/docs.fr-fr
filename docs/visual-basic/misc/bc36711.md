# <a name="argument-matching-parameter-parametername-narrows-to-typename"></a>Argument qui correspond au paramètre '\<nom_paramètre >' passe à '\<nom_type >'
Vous avez effectué un appel à une méthode surchargée, mais le compilateur ne peut pas trouver de méthode qui peut être appelée sans conversion restrictive. Une *conversion restrictive* modifie une valeur en un type de données qui peut ne pas pouvoir contenir certaines des valeurs possibles.  
  
 **ID d’erreur :** BC36711
