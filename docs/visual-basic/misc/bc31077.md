# <a name="assemblyname-is-an-assembly-and-cannot-be-referenced-as-a-module"></a>'\<nom_assembly >' est un assembly et ne peut pas être référencé en tant que module
Une tentative a été effectuée d’utiliser un assembly en tant que module.  
  
 **ID d’erreur :** BC31077  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Modifiez la référence à un module.
