# <a name="qualifiedmethod-is-not-accessible-in-this-context-because-it-is-modifier"></a>«\<méthode_qualifiée >' n’est pas accessible dans ce contexte, car il est '\<modificateur > »
Vous avez tenté d’accéder à une méthode qui n’est pas accessible dans ce contexte, car un modificateur lui a été appliqué.  
  
 **ID d’erreur :** BC30390  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Supprimez le modificateur.
