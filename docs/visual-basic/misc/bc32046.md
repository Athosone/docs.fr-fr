# <a name="new-cannot-be-used-on-a-type-parameter-that-does-not-have-a-new-constraint"></a><span data-ttu-id="06975-102">’New’ ne peut pas être utilisé pour un paramètre de type qui n’a pas de contrainte ’New’</span><span class="sxs-lookup"><span data-stu-id="06975-102">'New' cannot be used on a type parameter that does not have a 'New' constraint</span></span>
<span data-ttu-id="06975-103">Une instruction de déclaration utilise un [nouvel opérateur](../../visual-basic/language-reference/operators/new-operator.md) clause qui spécifie un paramètre de type en tant que le type doit être créé et le paramètre de type est déclaré sans un `New` contrainte.</span><span class="sxs-lookup"><span data-stu-id="06975-103">A declaration statement uses a [New Operator](../../visual-basic/language-reference/operators/new-operator.md) clause specifying a type parameter as the type to be created, and the type parameter is declared without a `New` constraint.</span></span>  
  
 <span data-ttu-id="06975-104">Une *contrainte* appliquée à un paramètre de type impose des exigences à tous les arguments de type passés à ce paramètre de type quand le type générique est créé.</span><span class="sxs-lookup"><span data-stu-id="06975-104">A *constraint* on a type parameter imposes a requirement on any type argument passed to that type parameter when the generic type is created.</span></span> <span data-ttu-id="06975-105">La contrainte `New` spécifie que l’argument de type doit exposer un constructeur sans paramètre accessible par le code de création.</span><span class="sxs-lookup"><span data-stu-id="06975-105">The `New` constraint specifies that the type argument must expose a parameterless constructor that the creating code can access.</span></span> <span data-ttu-id="06975-106">C’est ce qui permet à une clause `New` d’une instruction de déclaration de créer une instance de ce type.</span><span class="sxs-lookup"><span data-stu-id="06975-106">This is what allows a `New` clause in a declaration statement to create an instance of that type.</span></span>  
  
 <span data-ttu-id="06975-107">**ID d’erreur :** BC32046</span><span class="sxs-lookup"><span data-stu-id="06975-107">**Error ID:** BC32046</span></span>
