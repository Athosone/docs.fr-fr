# <a name="argument-argumentname-must-be-greater-than-or-equal-to-1"></a>Argument '\<nom_argument >' doit être supérieur ou égal à 1
La valeur d’un argument est inférieure à 1.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Vérifiez que la valeur de l’argument est comprise dans la plage valide.  
  
2. Si l’argument est fourni par un calcul, vérifiez les valeurs traitées par le calcul.
