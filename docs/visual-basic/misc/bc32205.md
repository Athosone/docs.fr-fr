# <a name="withevents-variable-name-length-cannot-exceed-1019-characters"></a>La longueur du nom de la variable WithEvents ne peut pas dépasser 1 019 caractères.
Le nom ou l’identificateur de chaque variable déclarée à l’aide de `WithEvents` est limité à 1 019 caractères.  
  
 **ID d’erreur :** BC32205  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Réduisez la longueur de l’identificateur.
