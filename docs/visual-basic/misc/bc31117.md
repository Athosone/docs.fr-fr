# <a name="raiseevent-declaration-must-end-with-a-matching-end-raiseevent"></a>La déclaration 'RaiseEvent' doit se terminer par un 'End RaiseEvent' correspondant
Une déclaration `RaiseEvent` doit se terminer par une instruction `End RaiseEvent` .  
  
 **ID d’erreur :** BC31117  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que la déclaration `RaiseEvent` se termine par une instruction `End RaiseEvent` .
