# <a name="method-in-a-structure-cannot-be-declared-protected-or-protected-friend"></a>Une méthode dans une structure ne peut pas être déclarée 'Protected' ni 'Protected Friend'
Une méthode dans une structure a été déclarée en tant que `Protected` ou `Protected Friend`.  
  
 **ID d’erreur :** BC31067  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Déclarez les méthodes à l’intérieur de structures en tant que `Public` ou `Private`.
