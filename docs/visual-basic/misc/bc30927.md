# <a name="mustoverride-cannot-be-specified-on-procedurename-because-it-is-in-a-partial-type-that-is-declared-notinheritable-in-another-partial-definition"></a><span data-ttu-id="59cb1-102">'MustOverride' ne peut pas être spécifié sur '\<nom_procédure >', car il se trouve dans un type partiel déclaré 'NotInheritable' dans une autre définition partielle</span><span class="sxs-lookup"><span data-stu-id="59cb1-102">'MustOverride' cannot be specified on '\<procedurename>' because it is in a partial type that is declared 'NotInheritable' in another partial definition</span></span>
<span data-ttu-id="59cb1-103">Une procédure ou une propriété est déclarée comme `MustOverride` dans une classe définie dans plusieurs déclarations partielles, mais l’une des définitions partielles spécifie `NotInheritable` pour la classe.</span><span class="sxs-lookup"><span data-stu-id="59cb1-103">A procedure or property is declared as `MustOverride` within a class that is defined in multiple partial declarations, but one of the partial definitions specifies `NotInheritable` for the class.</span></span>  
  
 <span data-ttu-id="59cb1-104">Quand vous divisez la définition d’une classe en plusieurs déclarations partielles, le compilateur traite la classe comme l’union de toutes ses déclarations partielles.</span><span class="sxs-lookup"><span data-stu-id="59cb1-104">When you divide the definition of a class among several partial declarations, the compiler treats the class as the union of all its partial declarations.</span></span> <span data-ttu-id="59cb1-105">Cela s’applique non seulement aux membres, mais aussi à l’implémentation, à l’héritage et au niveau d’accès.</span><span class="sxs-lookup"><span data-stu-id="59cb1-105">This applies not only to the members but also to the implementation, inheritance, and access level.</span></span>  
  
 <span data-ttu-id="59cb1-106">Pour substituer une procédure ou une propriété, une classe doit en hériter d’une classe de base.</span><span class="sxs-lookup"><span data-stu-id="59cb1-106">To override a procedure or property, a class must inherit it from a base class.</span></span> <span data-ttu-id="59cb1-107">Par conséquent, vous devez spécifier `MustOverride` pour la classe afin de spécifier `MustInherit` pour une procédure ou une propriété dans une classe de base.</span><span class="sxs-lookup"><span data-stu-id="59cb1-107">Therefore, to specify `MustOverride` for a procedure or property in a base class, you must specify `MustInherit` for the class.</span></span> <span data-ttu-id="59cb1-108">Ces mots clés étant mutuellement contradictoires, vous ne pouvez pas spécifier `MustInherit` et `NotInheritable` pour la même classe.</span><span class="sxs-lookup"><span data-stu-id="59cb1-108">Because they are mutually contradictory, you cannot specify both `MustInherit` and `NotInheritable` for the same class.</span></span>  
  
 <span data-ttu-id="59cb1-109">**ID d’erreur :** BC30927</span><span class="sxs-lookup"><span data-stu-id="59cb1-109">**Error ID:** BC30927</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="59cb1-110">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="59cb1-110">To correct this error</span></span>  
  
- <span data-ttu-id="59cb1-111">Si la propriété ou la procédure doit être substituée, supprimez le mot clé `NotInheritable` de la déclaration partielle dans laquelle il apparaît.</span><span class="sxs-lookup"><span data-stu-id="59cb1-111">If the property or procedure must be overridden, then remove the `NotInheritable` keyword from the partial declaration in which it appears.</span></span>  
  
- <span data-ttu-id="59cb1-112">Si la classe doit avoir la valeur `NotInheritable`, supprimez le mot clé `MustOverride` de la procédure ou la propriété.</span><span class="sxs-lookup"><span data-stu-id="59cb1-112">If the class must be `NotInheritable`, then remove the `MustOverride` keyword from the procedure or property.</span></span> <span data-ttu-id="59cb1-113">Vous ne pouvez pas le substituer, car vous ne pouvez pas hériter de la classe.</span><span class="sxs-lookup"><span data-stu-id="59cb1-113">You cannot override it because you cannot inherit the class.</span></span>
