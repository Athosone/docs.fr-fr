# <a name="multi-dimensional-array-cannot-be-converted-to-an-expression-tree"></a><span data-ttu-id="19935-102">Impossible de convertir un tableau multidimensionnel en une arborescence d’expression</span><span class="sxs-lookup"><span data-stu-id="19935-102">Multi-dimensional array cannot be converted to an expression tree</span></span>
<span data-ttu-id="19935-103">La plupart des expressions peuvent être converties en arborescences d’expression, mais pas les tableaux multidimensionnels.</span><span class="sxs-lookup"><span data-stu-id="19935-103">Most expressions can be converted to expression trees, but multi-dimensional arrays cannot be.</span></span> <span data-ttu-id="19935-104">Par exemple, le code suivant génère cette erreur :</span><span class="sxs-lookup"><span data-stu-id="19935-104">For example, the following code causes this error:</span></span>  
  
```vb  
Module Module1  
    Sub Main()  
        '' A multi-dimensional array cannot be converted.  
        'Dim expTree As Expressions.Expression(Of Func(Of Object)) = _  
        '    Function() New Integer(1, 1) {{1, 2}, {2, 3}}  
  
        ' A one-dimensional array can be converted.  
        Dim expTree2 As Expressions.Expression(Of Func(Of Object)) = _  
            Function() New Integer() {1, 2, 3}  
    End Sub  
End Module  
```
