# <a name="membername-is-already-declared-by-procedurename-which-was-generated-for-this-containername"></a>«\<nom_membre >' est déjà déclaré par '\<nom_procédure >', qui a été généré pour ce \<containername >

Un membre a été déclaré qui porte le même nom qu’un membre est déclaré par le common language runtime.

**ID d’erreur :** BC30733

## <a name="to-correct-this-error"></a>Pour corriger cette erreur

- Modifiez le nom du membre pour supprimer le conflit.
