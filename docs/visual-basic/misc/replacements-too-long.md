# <a name="replacements-too-long"></a>Remplacements trop longs
Vous avez spécifié un remplacement qui dépasse la longueur autorisée.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Consultez la documentation du composant pour en savoir plus sur les restrictions de longueur.
