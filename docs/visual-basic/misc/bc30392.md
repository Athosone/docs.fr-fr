# <a name="catch-cannot-catch-type-typename-because-it-is-not-systemexception-or-a-class-that-inherits-from-systemexception"></a>'Catch' ne peut pas intercepter le type '\<nom_type >', car il n’est pas 'System.Exception' ou une classe qui hérite de 'System.Exception'
`Catch` peut seulement intercepter des exceptions, et vous avez tenté d’intercepter un élément qui n’est pas dérivé d’une exception.  
  
 **ID d’erreur :** BC30392  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Supprimez l’instruction `Catch` ou remplacez la cible de `Catch` par une exception réelle.
