# <a name="argument-access-is-not-valid-valid-values-for-append-mode-are-openaccesswrite-and-openaccessdefault"></a>L’argument 'Access' n’est pas valide. Les valeurs valides pour le mode Append sont 'OpenAccess.Write' et 'OpenAccess.Default'
Une valeur d’accès n’est pas valide en mode Append.  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Remplacez la valeur par `OpenAccess.Write` ou `OpenAccess.Default`.
