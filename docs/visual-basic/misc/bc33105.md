# <a name="if-operands-cannot-be-named-arguments"></a><span data-ttu-id="39f26-102">Les opérandes 'If' ne peuvent pas être des arguments nommés</span><span class="sxs-lookup"><span data-stu-id="39f26-102">'If' operands cannot be named arguments</span></span>
<span data-ttu-id="39f26-103">L’utilisation d’arguments nommés dans les opérandes de l’opérateur `If` n’est pas valide.</span><span class="sxs-lookup"><span data-stu-id="39f26-103">Using named arguments in the operands of the `If` operator is not valid.</span></span> <span data-ttu-id="39f26-104">L’exemple suivant génère cette erreur :</span><span class="sxs-lookup"><span data-stu-id="39f26-104">The following example causes this error:</span></span>  
  
```  
Dim i As Integer  
Dim result As String  
' Not valid.  
' result = (If(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="39f26-105">Il diffère de la fonction `IIf` qui autorise des arguments nommés, comme indiqué dans le code suivant :</span><span class="sxs-lookup"><span data-stu-id="39f26-105">This differs from the `IIf` function, which does allow named arguments, as shown in the following code:</span></span>  
  
```  
' Valid.  
IIf(i > 0, TruePart:="positive", FalsePart:="not positive")  
```  
  
 <span data-ttu-id="39f26-106">**ID d’erreur :** BC33105</span><span class="sxs-lookup"><span data-stu-id="39f26-106">**Error ID:** BC33105</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="39f26-107">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="39f26-107">To correct this error</span></span>

    ```  
    result = If(i > 0, "positive", "not positive")  
    ```
