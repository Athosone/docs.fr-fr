# <a name="statement-is-not-valid-in-an-interface"></a>L’instruction n’est pas valide dans une interface
Une instruction non valide a été détectée dans l’interface.  
  
 **ID d’erreur :** BC31041  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que les instructions de l’interface définissent des éléments, mais qu’elles ne fournissent pas d’implémentation.
