# <a name="type-declaration-characters-are-not-valid-in-this-context"></a>Les caractères de déclaration de type ne sont pas valides dans ce contexte
Vous avez tenté d’utiliser un caractère qui n’est pas valide dans ce contexte, tel que $.  
  
 **ID d’erreur :** BC30468  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
1. Supprimez le caractère.
