# <a name="date-constant-is-not-valid"></a>Constante de date non valide
Une constante de date est déclarée avec une date non valide.  
  
 **ID d’erreur :** BC31085  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Corrigez la date utilisée dans la constante.
