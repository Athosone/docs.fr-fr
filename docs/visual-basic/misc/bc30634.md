# <a name="statement-cannot-appear-within-a-property-body"></a>L’instruction ne peut pas apparaître dans le corps d’une propriété
L’instruction ne peut pas apparaître dans le corps d’une propriété. Fin de propriété supposée.  
  
 Une instruction inattendue a été rencontrée à l’intérieur d’une propriété.  
  
 **ID d’erreur :** BC30634  
  
## <a name="to-correct-this-error"></a>Pour corriger cette erreur  
  
- Vérifiez que votre propriété utilise la syntaxe correcte et se termine par une construction `End Property` .
