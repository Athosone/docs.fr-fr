# <a name="type-parameter-typeparametername1-must-have-either-a-new-constraint-or-a-structure-constraint-to-satisfy-the-new-constraint-for-type-parameter-typeparametername2"></a><span data-ttu-id="56004-102">Paramètre de type '\<nom_paramètre_type1 >' doit avoir une contrainte 'New' ou une contrainte 'Structure' pour satisfaire la contrainte 'New' pour le paramètre de type '\<nom_paramètre_type2 >'</span><span class="sxs-lookup"><span data-stu-id="56004-102">Type parameter '\<typeparametername1>' must have either a 'New' constraint or a 'Structure' constraint to satisfy the 'New' constraint for type parameter '\<typeparametername2>'</span></span>
<span data-ttu-id="56004-103">Une instruction construit un type générique en passant un paramètre de type qui n’est pas contraint pour satisfaire une contrainte `New` .</span><span class="sxs-lookup"><span data-stu-id="56004-103">A statement constructs a generic type passing a type parameter that is not constrained to satisfy a `New` constraint.</span></span>  
  
 <span data-ttu-id="56004-104">La contrainte `New` signifie que l’argument de type fourni à ce paramètre de type doit exposer un constructeur sans paramètre accessible au code qui crée des objets à partir de celui-ci.</span><span class="sxs-lookup"><span data-stu-id="56004-104">The `New` constraint means that the type argument supplied to that type parameter must expose a parameterless constructor accessible to the code that creates objects from it.</span></span> <span data-ttu-id="56004-105">Tous les types valeur ont des constructeurs sans paramètre, mais ce n’est pas le cas de tous les types référence.</span><span class="sxs-lookup"><span data-stu-id="56004-105">All value types have parameterless constructors, but not all reference types do.</span></span> <span data-ttu-id="56004-106">La contrainte `Structure` satisfait donc la contrainte `New` , mais la contrainte `Class` , ou un nom de classe ou d’interface, ne la satisfont pas.</span><span class="sxs-lookup"><span data-stu-id="56004-106">Therefore the `Structure` constraint satisfies the `New` constraint, but the `Class` constraint, or a class or interface name, does not.</span></span>  
  
 <span data-ttu-id="56004-107">Les instructions suivantes peuvent générer cette erreur.</span><span class="sxs-lookup"><span data-stu-id="56004-107">The following statements can generate this error.</span></span>  
  
```  
Public Class c1(Of t As New)  
End Class  
Public Class c2(Of u)  
    Public testObject As New c1(Of u)  
End Class  
```  
  
 <span data-ttu-id="56004-108">Quand la classe `c2` crée un objet à partir de `c1`, le paramètre de type `u` ne satisfait pas la contrainte `New` appliquée au paramètre de type `t`.</span><span class="sxs-lookup"><span data-stu-id="56004-108">When class `c2` creates an object from `c1`, type parameter `u` does not satisfy the `New` constraint on type parameter `t`.</span></span>  
  
 <span data-ttu-id="56004-109">**ID d’erreur :** BC32084</span><span class="sxs-lookup"><span data-stu-id="56004-109">**Error ID:** BC32084</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="56004-110">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="56004-110">To correct this error</span></span>

