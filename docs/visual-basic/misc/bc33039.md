 <span data-ttu-id="0f2c7-104">La variable de compteur doit être d’un type qui prend en charge l’ajout (`+`) et la soustraction (`-`) d’opérateurs qui fonctionnent entièrement sur leur type conteneur.</span><span class="sxs-lookup"><span data-stu-id="0f2c7-104">The counter variable must be of a type that supports addition (`+`) and subtraction (`-`) operators that operate completely on their containing type.</span></span> <span data-ttu-id="0f2c7-105">Cela signifie que les opérandes et la valeur de retour doivent être du même type que celui de la variable de compteur.</span><span class="sxs-lookup"><span data-stu-id="0f2c7-105">This means both of the operands and the return value must be of the type of the counter variable.</span></span>
