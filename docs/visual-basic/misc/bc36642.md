# <a name="option-strict-on-requires-each-lambda-expression-parameter-to-be-declared-with-an-as-clause-if-its-type-cannot-be-inferred"></a><span data-ttu-id="e0664-102">Chaque paramètre d’expression lambda de Option Strict On doit être déclaré avec une clause ’As’ si son type ne peut pas être déduit</span><span class="sxs-lookup"><span data-stu-id="e0664-102">Option Strict On requires each lambda expression parameter to be declared with an 'As' clause if its type cannot be inferred</span></span>
<span data-ttu-id="e0664-103">Vous avez déclaré un paramètre dans une expression lambda sans utiliser de clause `As` , avec `Option Strict` activé.</span><span class="sxs-lookup"><span data-stu-id="e0664-103">You have declared a parameter in a lambda expression without using an `As` clause, with `Option Strict` on.</span></span>  
  
```  
' Not valid when Option Strict is on.  
' Dim increment1 = Function (n) n + 1  
```  
  
 <span data-ttu-id="e0664-104">La déclaration précédente est valide si le type de `n` peut être déduit.</span><span class="sxs-lookup"><span data-stu-id="e0664-104">The previous declaration is valid if the type of `n` can be inferred.</span></span> <span data-ttu-id="e0664-105">Par exemple, si vous assignez l’expression lambda précédente à un délégué de fonction, `Del`:</span><span class="sxs-lookup"><span data-stu-id="e0664-105">For example, if you are assigning the previous lambda expression to a function delegate, `Del`:</span></span>  
  
```  
Delegate Function Del(ByVal p As Integer) As Integer  
```  
  
 <span data-ttu-id="e0664-106">Le type de `n` peut désormais être déduit du paramètre `p`:</span><span class="sxs-lookup"><span data-stu-id="e0664-106">Now the type of `n` can be inferred from parameter `p`:</span></span>  
  
```  
Dim increment2 as Del = Function(n) n + 1  
```  
  
 <span data-ttu-id="e0664-107">**ID d’erreur :** BC36642</span><span class="sxs-lookup"><span data-stu-id="e0664-107">**Error ID:** BC36642</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="e0664-108">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="e0664-108">To correct this error</span></span>  
  
- <span data-ttu-id="e0664-109">Ajoutez une clause `As` à la déclaration du paramètre :</span><span class="sxs-lookup"><span data-stu-id="e0664-109">Add an `As` clause to the parameter declaration:</span></span>  
  
    ```  
    Dim increment3 = Function (n As Integer) n + 1  
    ```
