### <a name="description"></a><span data-ttu-id="478b3-105">Description</span><span class="sxs-lookup"><span data-stu-id="478b3-105">Description</span></span>  
 <span data-ttu-id="478b3-106">Cet exemple déclare deux espaces de noms.</span><span class="sxs-lookup"><span data-stu-id="478b3-106">This example declares two namespaces.</span></span> <span data-ttu-id="478b3-107">Il spécifie que le `http://www.adventure-works.com` espace de noms a le préfixe `aw`et que le `www.fourthcoffee.com` espace de noms a le préfixe `fc`.</span><span class="sxs-lookup"><span data-stu-id="478b3-107">It specifies that the `http://www.adventure-works.com` namespace has the prefix `aw`, and that the `www.fourthcoffee.com` namespace has the prefix of `fc`.</span></span>  
  
### <a name="code"></a><span data-ttu-id="478b3-108">Code</span><span class="sxs-lookup"><span data-stu-id="478b3-108">Code</span></span>  
  
```vb  
Imports <xmlns:aw="http://www.adventure-works.com">  
Imports <xmlns:fc="www.fourthcoffee.com">  
  
Module Module1  
  
    Sub Main()  
        Dim root As XElement = _  
            <aw:Root>  
                <fc:Child>  
                    <aw:DifferentChild>other content</aw:DifferentChild>  
                </fc:Child>  
                <aw:Child2>c2 content</aw:Child2>  
                <fc:Child3>c3 content</fc:Child3>  
            </aw:Root>  
        Console.WriteLine(root)  
    End Sub  
  
End Module  
```
