## <a name="example"></a><span data-ttu-id="7aa7b-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="7aa7b-107">Example</span></span>  
 <span data-ttu-id="7aa7b-108">Le code suivant génère l’erreur CS0845 :</span><span class="sxs-lookup"><span data-stu-id="7aa7b-108">The following code generates CS0845:</span></span>  
  
```csharp  
// cs0845.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace ConsoleApplication1  
{  
    class Program  
    {  
        static void Main(string[] args)  
        {  
            Expression<Func<object>> e = () => null ?? null; // CS0845  
            // Try the following line instead.  
            // Expression<Func<object>> e = () => (object)null ?? null;  
        }  
    }  
}  
```
