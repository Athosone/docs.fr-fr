# <a name="default-c-reference"></a>default (référence C#)

Le mot clé `default` peut être utilisé de deux façons :
- [l’instruction `switch`](switch.md) pour spécifier l’étiquette par défaut ;
- une [expression de valeur par défaut](../../programming-guide/statements-expressions-operators/default-value-expressions.md) pour produire la valeur par défaut d’un type.
