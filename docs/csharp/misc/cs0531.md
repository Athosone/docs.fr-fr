# <a name="compiler-error-cs0531"></a><span data-ttu-id="8e6f9-102">Erreur du compilateur CS0531</span><span class="sxs-lookup"><span data-stu-id="8e6f9-102">Compiler Error CS0531</span></span>
<span data-ttu-id="8e6f9-103">'membre' : les membres d’interface ne peuvent pas avoir de définition</span><span class="sxs-lookup"><span data-stu-id="8e6f9-103">'member' : interface members cannot have a definition</span></span>  
  
 <span data-ttu-id="8e6f9-104">Les méthodes déclarées dans une [interface](../../csharp/language-reference/keywords/interface.md) doivent être implémentées dans une classe qui en hérite et non dans l’interface elle-même.</span><span class="sxs-lookup"><span data-stu-id="8e6f9-104">Methods that are declared in an [interface](../../csharp/language-reference/keywords/interface.md) must be implemented in a class that inherits from it and not in the interface itself.</span></span>  
  
 <span data-ttu-id="8e6f9-105">L’exemple suivant génère l’erreur CS0531 :</span><span class="sxs-lookup"><span data-stu-id="8e6f9-105">The following sample generates CS0531:</span></span>  
  
```csharp  
// CS0531.cs  
namespace x  
{  
   public interface clx  
   {  
      int xclx()   // CS0531, cannot define xclx  
      // Try the following declaration instead:  
      // int xclx();  
      {  
         return 0;  
      }  
   }  
  
   public class cly  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
