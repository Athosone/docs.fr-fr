# <a name="compiler-error-cs0177"></a>Erreur du compilateur CS0177
Le paramètre de sortie 'parameter' doit être assigné avant que le contrôle ne quitte la méthode actuelle  
  
 Un paramètre marqué avec le mot clé [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) n’a pas de valeur assignée dans le corps de la méthode. Pour plus d’informations, consultez [passage de paramètres](../../csharp/programming-guide/classes-and-structs/passing-parameters.md)  
  
 L’exemple suivant génère l’erreur CS0177 :  
  
```csharp  
// CS0177.cs  
public class MyClass  
{  
   public static void Foo(out int i)   // CS0177  
   {  
   // uncomment the following line to resolve this error  
   //   i = 0;  
   }  
  
   public static void Main()  
   {  
       int x = -1;  
       Foo(out x);  
   }  
}  
```
