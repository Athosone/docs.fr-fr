# <a name="compiler-error-cs1555"></a>Erreur du compilateur CS1555
Impossible de trouver 'classe' spécifié pour la méthode Main  
  
 Une classe a été spécifiée dans l’option du compilateur [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) , mais le nom de la classe est introuvable dans le code source.
