## <a name="example"></a><span data-ttu-id="3023a-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="3023a-106">Example</span></span>  
 <span data-ttu-id="3023a-107">L’exemple suivant génère l’erreur CS0305 :</span><span class="sxs-lookup"><span data-stu-id="3023a-107">The following sample generates CS0305.</span></span>  
  
```csharp

class MyClass  
{  
   public static void Main()

      MyList<MyClass, MyClass> list1 = new MyList<MyClass>();   // CS0305  
      MyList<MyClass> list2 = new MyList<MyClass>();   // OK  
   }  
}  
```
