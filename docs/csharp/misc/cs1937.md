## <a name="example"></a><span data-ttu-id="eaa55-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="eaa55-110">Example</span></span>  
 <span data-ttu-id="eaa55-111">L’exemple suivant génère l’erreur CS1937.</span><span class="sxs-lookup"><span data-stu-id="eaa55-111">The following example generates CS1937.</span></span>  
  
```csharp  
// cs1937.cs  
using System.Linq;  
class Test  
{  
    static void Main()  
    {  
        int[] sourceA = { 1, 2, 3, 4, 5 };  
        int[] sourceB = { 3, 4, 5, 6, 7 };  
  
        var query = from a in sourceA  
                    join b in sourceB on b equals a // CS1937  
                    // Try the following line instead.  
                    //join b in sourceB on a equals b  
                    select new { a, b };  
    }  
}  
```
