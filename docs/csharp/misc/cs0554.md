<span data-ttu-id="8a9c6-103">'routine de conversion' : conversion définie par l’utilisateur vers/de la classe dérivée</span><span class="sxs-lookup"><span data-stu-id="8a9c6-103">'conversion routine' : user defined conversion to/from derived class</span></span>  
  
 <span data-ttu-id="8a9c6-104">Les conversions définies par l’utilisateur en valeurs de classe dérivée ne sont pas autorisées. Ce type d’opérateur n’est pas nécessaire.</span><span class="sxs-lookup"><span data-stu-id="8a9c6-104">User-defined conversions to values of a derived class are not allowed; you do not need such an operator.</span></span>  
  
 <span data-ttu-id="8a9c6-105">Pour plus d’informations sur les conversions définies par l’utilisateur, consultez le chapitre 6 de la spécification du langage C#.</span><span class="sxs-lookup"><span data-stu-id="8a9c6-105">See chapter 6 in the C# language specification for more information on user-defined conversions.</span></span>  
  
 <span data-ttu-id="8a9c6-106">L’exemple suivant génère l’erreur CS0554 :</span><span class="sxs-lookup"><span data-stu-id="8a9c6-106">The following sample generates CS0554:</span></span>  
  
```csharp  
// CS0554.cs  
namespace x  
{  
   public class ii  
   {  
      // delete the conversion routine to resolve CS0554  
      public static implicit operator ii(a aa) // CS0554  
      {  
         return new ii();  
      }  
   }  
  
   public class a : ii  
   {  
      public static void Main()  
      {  
      }  
   }  
}  
```
