# <a name="compiler-error-cs0744"></a><span data-ttu-id="3cf85-102">Erreur du compilateur CS0744</span><span class="sxs-lookup"><span data-stu-id="3cf85-102">Compiler Error CS0744</span></span>
<span data-ttu-id="3cf85-103">Mot clé contextuel 'equals' attendu</span><span class="sxs-lookup"><span data-stu-id="3cf85-103">Expected contextual keyword 'equals'</span></span>  
  
 <span data-ttu-id="3cf85-104">Le modèle pour une clause `join` est `join`...`in`...`on`...`equals`, comme illustré dans cet exemple :</span><span class="sxs-lookup"><span data-stu-id="3cf85-104">The pattern for a `join` clause is `join`...`in`...`on`...`equals`, as shown in this example:</span></span>

```csharp  
            join y in array2 on x equals y
```  

## <a name="example"></a><span data-ttu-id="3cf85-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="3cf85-107">Example</span></span>  
 <span data-ttu-id="3cf85-108">L’exemple suivant génère l’erreur CS0744 :</span><span class="sxs-lookup"><span data-stu-id="3cf85-108">The following code generates CS0744:</span></span>

```csharp  
using System;  
using System.Linq;  
  
public class C  
{  
    public static int Main()  
    {  
        int[] array1 = { 1, 2, 3 ,4, 5, 6 };  
        int[] array2 = { 5, 6, 7, 8, 9 };  
        var c = from x in array1  
                join y in array2 on x y // CS0744  
                select x;  
        return 1;  
    }  
}  
```
