## <a name="example"></a><span data-ttu-id="5f936-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="5f936-106">Example</span></span>  
 <span data-ttu-id="5f936-107">L’exemple suivant génère l’erreur CS0540.</span><span class="sxs-lookup"><span data-stu-id="5f936-107">The following sample generates CS0540.</span></span>  
  
```csharp  
// CS0540.cs  
interface I  
{  
   void m();  
}  
  
public class Clx  
{  
   void I.m() {}   // CS0540  
}  
  
// OK  
public class Cly : I  
{  
   void I.m() {}  
   public static void Main() {}  
}  
```
