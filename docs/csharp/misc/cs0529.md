<span data-ttu-id="4eab0-103">L’interface héritée 'interface1' provoque un cycle dans la hiérarchie des interfaces de 'interface2'</span><span class="sxs-lookup"><span data-stu-id="4eab0-103">Inherited interface 'interface1' causes a cycle in the interface hierarchy of 'interface2'</span></span>  
  
 <span data-ttu-id="4eab0-104">La liste d’héritage pour une [interface](../../csharp/language-reference/keywords/interface.md) comprend une référence directe ou indirecte à elle-même.</span><span class="sxs-lookup"><span data-stu-id="4eab0-104">The inheritance list for an [interface](../../csharp/language-reference/keywords/interface.md) includes a direct or indirect reference to itself.</span></span> <span data-ttu-id="4eab0-105">Une interface ne peut pas hériter d’elle-même.</span><span class="sxs-lookup"><span data-stu-id="4eab0-105">An interface cannot inherit from itself.</span></span>  
  
 <span data-ttu-id="4eab0-106">L’exemple suivant génère l’erreur CS0529 :</span><span class="sxs-lookup"><span data-stu-id="4eab0-106">The following sample generates CS0529:</span></span>  
  
```csharp  
// CS0529.cs  
namespace x  
{  
   public interface a  
   {  
   }  
  
   public interface b : a, c  
   {  
   }  
  
   public interface c : b   // CS0529, b inherits from c  
   {  
   }  
}  
```
