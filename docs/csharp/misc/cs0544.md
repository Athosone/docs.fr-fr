# <a name="compiler-error-cs0544"></a><span data-ttu-id="985d3-102">Erreur du compilateur CS0544</span><span class="sxs-lookup"><span data-stu-id="985d3-102">Compiler Error CS0544</span></span>
<span data-ttu-id="985d3-103">'substitution_propriété' : substitution impossible, car 'non_propriété' n’est pas une propriété</span><span class="sxs-lookup"><span data-stu-id="985d3-103">'property override': cannot override because 'non-property' is not a property</span></span>  
  
 <span data-ttu-id="985d3-104">Vous avez tenté de substituer un type de données autre qu’une propriété en tant que [propriété](../../csharp/programming-guide/classes-and-structs/properties.md), ce qui n’est pas autorisé.</span><span class="sxs-lookup"><span data-stu-id="985d3-104">An attempt was made to override a nonproperty data type as a [property](../../csharp/programming-guide/classes-and-structs/properties.md), which is not allowed.</span></span>  
  
 <span data-ttu-id="985d3-105">L’exemple suivant génère l’erreur CS0544 :</span><span class="sxs-lookup"><span data-stu-id="985d3-105">The following sample generates CS0544:</span></span>  
  
```csharp  
// CS0544.cs  
// compile with: /target:library  
public class a  
{  
   public int i;  
}  
  
public class b : a  
{  
   public override int i {   // CS0544  
   // try the following line instead  
   // public new int i {  
      get  
      {  
         return 0;  
      }  
   }  
}  
```
