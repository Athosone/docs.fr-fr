<span data-ttu-id="c5d61-103">Les structures ne peuvent pas contenir de constructeurs exempts de paramètres explicites</span><span class="sxs-lookup"><span data-stu-id="c5d61-103">Structs cannot contain explicit parameterless constructors</span></span>  
  
 <span data-ttu-id="c5d61-104">Chaque [struct](../../csharp/language-reference/keywords/struct.md) a déjà un constructeur sans paramètre qui initialise l’objet à zéro.</span><span class="sxs-lookup"><span data-stu-id="c5d61-104">Each [struct](../../csharp/language-reference/keywords/struct.md) already has a parameterless constructor that initializes the object to zero.</span></span> <span data-ttu-id="c5d61-105">Les constructeurs que vous pouvez créer pour un struct doivent donc accepter un ou plusieurs paramètres.</span><span class="sxs-lookup"><span data-stu-id="c5d61-105">Therefore, the constructors that you can create for a struct must take one or more parameters.</span></span>  
  
 <span data-ttu-id="c5d61-106">L’exemple suivant génère l’erreur CS0568 :</span><span class="sxs-lookup"><span data-stu-id="c5d61-106">The following sample generates CS0568:</span></span>

```csharp  
   public int field1;  
   public ClassY(){}   // CS0568, cannot have no param constructor

   // public ClassY(int i)  
   // {  
   //    field1 = i;  
   // }  
}

{  
   public static void Main()  
   {  
   }  
}  
```
