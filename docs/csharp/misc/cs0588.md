# <a name="compiler-error-cs0588"></a>Erreur du compilateur CS0588
Erreur interne du compilateur : étape 'LEX'  
  
 Essayez de déterminer si le compilateur échoue en raison de son incapacité à analyser une syntaxe inattendue. Si ce n’est pas le cas, essayez de [nous contacter](/visualstudio/ide/talk-to-us).
