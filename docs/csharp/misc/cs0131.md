 <span data-ttu-id="49c4d-104">Dans une instruction d’assignation, la valeur de la partie droite est assignée à la partie gauche.</span><span class="sxs-lookup"><span data-stu-id="49c4d-104">In an assignment statement, the value of the right-hand side is assigned to the left-hand side.</span></span> <span data-ttu-id="49c4d-105">La partie gauche doit être une variable, une propriété ou un indexeur.</span><span class="sxs-lookup"><span data-stu-id="49c4d-105">The left-hand side must be a variable, property, or indexer.</span></span>  
  
 <span data-ttu-id="49c4d-106">Pour corriger cette erreur, vérifiez que tous les opérateurs sont dans la partie droite et que la partie gauche correspond à une variable, une propriété ou un indexeur.</span><span class="sxs-lookup"><span data-stu-id="49c4d-106">To fix this error, make sure that all operators are on the right-hand side and that the left-hand side is a variable, property, or indexer.</span></span> <span data-ttu-id="49c4d-107">Pour plus d’informations, consultez [Instructions, expressions et opérateurs](../../csharp/programming-guide/statements-expressions-operators/index.md).</span><span class="sxs-lookup"><span data-stu-id="49c4d-107">For more information, see [Statements, Expressions, and Operators](../../csharp/programming-guide/statements-expressions-operators/index.md).</span></span>  
  
## <a name="example"></a><span data-ttu-id="49c4d-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="49c4d-108">Example</span></span>  
 <span data-ttu-id="49c4d-109">L’exemple suivant génère l’erreur CS0131.</span><span class="sxs-lookup"><span data-stu-id="49c4d-109">The following sample generates CS0131.</span></span>  
  
```csharp  
// CS0131.cs  
public class MyClass  
{  
    public int i = 0;  
    public void MyMethod()  
    {  
        i++ = 1;   // CS0131  
        // try the following line instead  
        // i = 1;  
    }  
    public static void Main() { }  
}  
```  
  
## <a name="example"></a><span data-ttu-id="49c4d-110">Exemple</span><span class="sxs-lookup"><span data-stu-id="49c4d-110">Example</span></span>  
 <span data-ttu-id="49c4d-111">Cette erreur peut également se produire si vous essayez d’exécuter des opérations arithmétiques dans la partie gauche d’un opérateur d’assignation, comme dans l’exemple suivant.</span><span class="sxs-lookup"><span data-stu-id="49c4d-111">This error can also occur if you attempt to perform arithmetic operations on the left hand side of an assignment operator, as in the following example.</span></span>

```csharp  
public class C  
{  
    public static int Main()  
    {  
        int a = 1, b = 2, c = 3;  
        if (a + b = c) // CS0131  
        // try this instead  
        // if (a + b == c)

        return 1;  
    }  
}  
```
