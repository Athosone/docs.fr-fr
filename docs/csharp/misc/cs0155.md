# <a name="compiler-error-cs0155"></a><span data-ttu-id="9eb76-102">Erreur du compilateur CS0155</span><span class="sxs-lookup"><span data-stu-id="9eb76-102">Compiler Error CS0155</span></span>
<span data-ttu-id="9eb76-103">Le type intercepté ou levé doit être dérivé de System.Exception</span><span class="sxs-lookup"><span data-stu-id="9eb76-103">The type caught or thrown must be derived from System.Exception</span></span>  
  
 <span data-ttu-id="9eb76-104">Une tentative a été effectuée de passer un type de données qui ne dérive pas de **System.Exception** à un bloc [catch](../../csharp/language-reference/keywords/try-catch.md) .</span><span class="sxs-lookup"><span data-stu-id="9eb76-104">An attempt was made to pass a data type that does not derive from **System.Exception** into a [catch](../../csharp/language-reference/keywords/try-catch.md) block.</span></span> <span data-ttu-id="9eb76-105">Seuls les types de données qui dérivent de **System.Exception** peuvent être passés à un bloc **catch** .</span><span class="sxs-lookup"><span data-stu-id="9eb76-105">Only data types that derive from **System.Exception** can be passed into a **catch** block.</span></span> <span data-ttu-id="9eb76-106">Pour plus d’informations, consultez [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) et [Exceptions et gestion des exceptions](../../csharp/programming-guide/exceptions/index.md).</span><span class="sxs-lookup"><span data-stu-id="9eb76-106">For more information, see [Exception Handling Statements](../../csharp/language-reference/keywords/exception-handling-statements.md) and [Exceptions and Exception Handling](../../csharp/programming-guide/exceptions/index.md).</span></span>  
  
 <span data-ttu-id="9eb76-107">L’exemple suivant génère l’erreur CS0155 :</span><span class="sxs-lookup"><span data-stu-id="9eb76-107">The following sample generates CS0155:</span></span>  
  
```csharp  
// CS0155.cs  
using System;  
  
namespace MyNamespace  
{  
    public class MyClass2  
    // try the following line instead  
    // public class MyClass2 : Exception  
    {  
    }  
    public class MyClass  
    {  
        public static void Main()  
        {  
            try  
            {  
            }  
            catch (MyClass2)   // CS0155, resolves if you derive MyClass2 from Exception  
            {  
            }  
        }  
    }  
}  
```
