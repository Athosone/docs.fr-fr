# <a name="compiler-error-cs0838"></a><span data-ttu-id="a3679-102">Erreur du compilateur CS0838</span><span class="sxs-lookup"><span data-stu-id="a3679-102">Compiler Error CS0838</span></span>

 <span data-ttu-id="a3679-104">Vous ne pouvez pas initialiser des tableaux multidimensionnels dans des arborescences d’expression à l’aide d’un initialiseur de tableau.</span><span class="sxs-lookup"><span data-stu-id="a3679-104">Multidimensional arrays in expression trees cannot be initialized by using an array initializer.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="a3679-105">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="a3679-105">To correct this error</span></span>  
  
1. <span data-ttu-id="a3679-106">Créez et initialisez le tableau avant de créer l’arborescence d’expression.</span><span class="sxs-lookup"><span data-stu-id="a3679-106">Create and initialize the array before creating the expression tree.</span></span>  
  
## <a name="example"></a><span data-ttu-id="a3679-107">Exemple</span><span class="sxs-lookup"><span data-stu-id="a3679-107">Example</span></span>  
 <span data-ttu-id="a3679-108">L’exemple suivant génère l’erreur CS0838 :</span><span class="sxs-lookup"><span data-stu-id="a3679-108">The following example generates CS0838:</span></span>  
  
```csharp  
// cs0838.cs  
using System;  
using System.Linq;  
using System.Linq.Expressions;  
  
namespace TestNamespace  
{  
    class Test  
    {  
        static int Main()  
        {  
  
            Expression<Func<int[,]>> expr =  
                () => new int[2, 2] { { 1, 2 }, { 3, 4 } }; // CS0838  
  
            // try the following 2 lines instead  
            int[,] nums = new int[2, 2] { { 1, 2 }, { 3, 4 } };  
            Expression<Func<int[,]>> expr2 = () => nums;   
  
            return 1;  
        }  
    }  
}  
```
