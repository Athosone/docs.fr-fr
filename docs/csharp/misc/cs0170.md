<span data-ttu-id="4a39e-103">Utilisation d’un champ potentiellement non assigné 'field'</span><span class="sxs-lookup"><span data-stu-id="4a39e-103">Use of possibly unassigned field 'field'</span></span>  
  
 <span data-ttu-id="4a39e-104">Un champ dans une structure a été utilisé sans être au préalable initialisé.</span><span class="sxs-lookup"><span data-stu-id="4a39e-104">A field in a structure was used without first being initialized.</span></span> <span data-ttu-id="4a39e-105">Pour résoudre ce problème, déterminez d’abord le champ qui a été initialisé, puis initialisez-le avant d’essayer d’y accéder.</span><span class="sxs-lookup"><span data-stu-id="4a39e-105">To solve this problem, first determine which field was uninitialized and then initialize it before you try to access it.</span></span> <span data-ttu-id="4a39e-106">Pour plus d’informations sur l’initialisation de structures, consultez [Structs](../../csharp/programming-guide/classes-and-structs/structs.md) et [utilisation de Structs](../../csharp/programming-guide/classes-and-structs/using-structs.md).</span><span class="sxs-lookup"><span data-stu-id="4a39e-106">For more information about initializing structs, see [Structs](../../csharp/programming-guide/classes-and-structs/structs.md) and [Using Structs](../../csharp/programming-guide/classes-and-structs/using-structs.md).</span></span>  
  
 <span data-ttu-id="4a39e-107">L’exemple suivant génère l’erreur CS0170 :</span><span class="sxs-lookup"><span data-stu-id="4a39e-107">The following sample generates CS0170:</span></span>  
  
```csharp  
// CS0170.cs  
public struct error  
{  
   public int i;  
}  
  
public class MyClass  
{  
   public static void Main()  
   {  
      error e;  
      // uncomment the next line to resolve this error  
      // e.i = 0;  
      System.Console.WriteLine( e.i );   // CS0170 because   
                                         //e.i was never assigned  
   }  
}  
```
