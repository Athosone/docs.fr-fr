 <span data-ttu-id="9b1b7-104">[sealed](../../csharp/language-reference/keywords/sealed.md) a été utilisé sur un membre qui n’était pas marqué avec [override](../../csharp/language-reference/keywords/override.md).</span><span class="sxs-lookup"><span data-stu-id="9b1b7-104">[sealed](../../csharp/language-reference/keywords/sealed.md) was used on a member that was not also marked with [override](../../csharp/language-reference/keywords/override.md).</span></span> <span data-ttu-id="9b1b7-105">Pour plus d’informations, consultez [Héritage](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span><span class="sxs-lookup"><span data-stu-id="9b1b7-105">For more information, see [Inheritance](../../csharp/programming-guide/classes-and-structs/inheritance.md).</span></span>
