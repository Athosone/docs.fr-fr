# <a name="compiler-error-cs1007"></a><span data-ttu-id="6d352-102">Erreur du compilateur CS1007</span><span class="sxs-lookup"><span data-stu-id="6d352-102">Compiler Error CS1007</span></span>
<span data-ttu-id="6d352-103">Accesseur de propriété déjà défini</span><span class="sxs-lookup"><span data-stu-id="6d352-103">Property accessor already defined</span></span>  
  
 <span data-ttu-id="6d352-104">Lorsque vous déclarez une [propriété](../../csharp/programming-guide/classes-and-structs/using-properties.md), vous devez également déclarer ses méthodes d’accesseur.</span><span class="sxs-lookup"><span data-stu-id="6d352-104">When you declare a [property](../../csharp/programming-guide/classes-and-structs/using-properties.md), you must also declare its accessor methods.</span></span> <span data-ttu-id="6d352-105">Toutefois, une propriété ne peut pas avoir plusieurs méthodes d’accesseur `get` ni plusieurs méthodes d’accesseur `set` .</span><span class="sxs-lookup"><span data-stu-id="6d352-105">However, a property cannot have more than one `get` accessor method or more than one `set` accessor method.</span></span>  
  
## <a name="example"></a><span data-ttu-id="6d352-106">Exemple</span><span class="sxs-lookup"><span data-stu-id="6d352-106">Example</span></span>  
 <span data-ttu-id="6d352-107">L’exemple suivant génère l’erreur CS1007 :</span><span class="sxs-lookup"><span data-stu-id="6d352-107">The following sample generates CS1007:</span></span>  
  
```csharp  
// CS1007.cs  
public class clx  
{  
    public int MyProperty  
    {  
        get  
        {  
            return 0;  
        }  
        get   // CS1007, this is the second get method  
        {  
            return 0;  
        }  
    }  
  
    public static void Main() {}  
}  
```
