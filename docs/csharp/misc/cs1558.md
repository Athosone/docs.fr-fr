# <a name="compiler-error-cs1558"></a><span data-ttu-id="6d8af-102">Erreur du compilateur CS1558</span><span class="sxs-lookup"><span data-stu-id="6d8af-102">Compiler Error CS1558</span></span>
<span data-ttu-id="6d8af-103">'class' ne dispose pas d’une méthode Main static appropriée</span><span class="sxs-lookup"><span data-stu-id="6d8af-103">'class' does not have a suitable static Main method</span></span>  
  
 <span data-ttu-id="6d8af-104">L’option du compilateur [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) a spécifié une classe dans laquelle rechercher une méthode **Main** .</span><span class="sxs-lookup"><span data-stu-id="6d8af-104">The [/main](../../csharp/language-reference/compiler-options/main-compiler-option.md) compiler option specified a class in which to look for a **Main** method.</span></span> <span data-ttu-id="6d8af-105">Toutefois, la méthode [Main](../../csharp/programming-guide/main-and-command-args/index.md) n’était pas définie correctement.</span><span class="sxs-lookup"><span data-stu-id="6d8af-105">However, the [Main](../../csharp/programming-guide/main-and-command-args/index.md) method was not defined correctly.</span></span>  
  
 <span data-ttu-id="6d8af-106">L’exemple suivant génère l’erreur CS1558 en raison d’un type de retour non valide.</span><span class="sxs-lookup"><span data-stu-id="6d8af-106">The following example generates CS1558 because of invalid return type.</span></span>  
  
```csharp  
// CS1558.cs  
// compile with: /main:MyNamespace.MyClass  
  
namespace MyNamespace  
{  
   public class MyClass  
   {  
      public static float Main()   
      {  
         return 0.0; // CS1558 because the return type is a float.  
      }  
   }  
}  
```
