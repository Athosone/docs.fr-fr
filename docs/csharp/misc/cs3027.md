## <a name="example"></a><span data-ttu-id="76e66-105">Exemple</span><span class="sxs-lookup"><span data-stu-id="76e66-105">Example</span></span>  
 <span data-ttu-id="76e66-106">L’exemple suivant contient une interface avec une méthode qui utilise un type non conforme CLS dans sa signature, ce qui rend le type non conforme CLS.</span><span class="sxs-lookup"><span data-stu-id="76e66-106">The following sample contains an interface with a method that uses a non-CLS compliant type in its signature, making the type non-CLS compliant.</span></span>

   void IMethod(uint i);
