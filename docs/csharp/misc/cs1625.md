# <a name="compiler-error-cs1625"></a><span data-ttu-id="eb9c4-102">Erreur du compilateur CS1625</span><span class="sxs-lookup"><span data-stu-id="eb9c4-102">Compiler Error CS1625</span></span>
<span data-ttu-id="eb9c4-103">Impossible de générer dans le corps d’une clause finally</span><span class="sxs-lookup"><span data-stu-id="eb9c4-103">Cannot yield in the body of a finally clause</span></span>  
  
 <span data-ttu-id="eb9c4-104">Une instruction yield n’est pas autorisée dans le corps d’une clause finally.</span><span class="sxs-lookup"><span data-stu-id="eb9c4-104">A yield statement is not allowed in the body of a finally clause.</span></span> <span data-ttu-id="eb9c4-105">Pour éviter cette erreur, placez l’instruction yield en dehors de la clause finally.</span><span class="sxs-lookup"><span data-stu-id="eb9c4-105">To avoid this error, move the yield statement out of the finally clause.</span></span>  
  
 <span data-ttu-id="eb9c4-106">L’exemple suivant génère l’erreur CS1625 :</span><span class="sxs-lookup"><span data-stu-id="eb9c4-106">The following sample generates CS1625:</span></span>  
  
```csharp  
// CS1625.cs  
using System.Collections;  
  
class C : IEnumerable  
{  
   public IEnumerator GetEnumerator()  
   {  
      try  
      {  
      }  
      finally  
      {  
        yield return this;  // CS1625  
      }  
   }  
}  
  
public class CMain  
{  
   public static void Main() { }  
}  
```
