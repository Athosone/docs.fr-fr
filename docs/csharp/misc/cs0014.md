# <a name="compiler-error-cs0014"></a>Erreur du compilateur CS0014
Le fichier requis 'fichier' est introuvable  
  
 Un fichier requis par le compilateur est introuvable sur le système. Vérifiez que le chemin est correct. Si le fichier est un fichier de système de Visual Studio, puis vous devrez peut-être réparer votre installation ou supprimer et réinstaller complètement de Visual Studio.
