# <a name="compiler-error-cs0601"></a><span data-ttu-id="bb2cc-102">Erreur du compilateur CS0601</span><span class="sxs-lookup"><span data-stu-id="bb2cc-102">Compiler Error CS0601</span></span>
<span data-ttu-id="bb2cc-103">L'attribut DllImport doit être spécifié sur une méthode marquée 'static' et 'extern'</span><span class="sxs-lookup"><span data-stu-id="bb2cc-103">The DllImport attribute must be specified on a method marked 'static' and 'extern'</span></span>  
  
 <span data-ttu-id="bb2cc-104">L’attribut `DllImport` a été utilisé sur une méthode qui n’utilise pas les mots clés d’accès corrects.</span><span class="sxs-lookup"><span data-stu-id="bb2cc-104">The `DllImport` attribute was used on a method that did not have the correct access keywords.</span></span>  
  
 <span data-ttu-id="bb2cc-105">L’exemple suivant génère l’erreur CS0601 :</span><span class="sxs-lookup"><span data-stu-id="bb2cc-105">The following sample generates CS0601:</span></span>  
  
```csharp  
// CS0601.cs  
using System.Runtime.InteropServices;  
using System.Text;  
  
public class C  
{  
   [DllImport("KERNEL32.DLL")]  
   extern int GetCurDirectory(int bufSize, StringBuilder buf);   // CS0601  
   // Try the following line instead:  
   // static extern int GetCurDirectory(int bufSize, StringBuilder buf);  
}  
  
public class MainClass  
{  
   public static void Main ()  
   {  
   }  
}  
```
