## <a name="example"></a><span data-ttu-id="0a01d-109">Exemple</span><span class="sxs-lookup"><span data-stu-id="0a01d-109">Example</span></span>  
 <span data-ttu-id="0a01d-110">Le code suivant génère l’erreur CS1947 :</span><span class="sxs-lookup"><span data-stu-id="0a01d-110">The following code generates CS1947:</span></span>

class Test  
{  
    static void Main()  
    {  
        int[] array = new int[] { 1, 2, 3, 4, 5 };
