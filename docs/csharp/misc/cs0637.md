# <a name="compiler-error-cs0637"></a><span data-ttu-id="e1655-102">Erreur du compilateur CS0637</span><span class="sxs-lookup"><span data-stu-id="e1655-102">Compiler Error CS0637</span></span>

<span data-ttu-id="e1655-103">L’attribut FieldOffset n’est pas autorisé sur des champs statiques ou const.</span><span class="sxs-lookup"><span data-stu-id="e1655-103">The FieldOffset attribute is not allowed on static or const fields.</span></span>
  
 <span data-ttu-id="e1655-104">L’attribut [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) ne peut pas être utilisé sur les champs marqués avec [static](../../csharp/language-reference/keywords/static.md) ou [const](../../csharp/language-reference/keywords/const.md).</span><span class="sxs-lookup"><span data-stu-id="e1655-104">The [FieldOffset](xref:System.Runtime.InteropServices.FieldOffsetAttribute) attribute cannot be used on fields marked with [static](../../csharp/language-reference/keywords/static.md) or [const](../../csharp/language-reference/keywords/const.md).</span></span>  
  
 <span data-ttu-id="e1655-105">L’exemple suivant génère l’erreur CS0637 :</span><span class="sxs-lookup"><span data-stu-id="e1655-105">The following sample generates CS0637:</span></span>  
  
```csharp  
// CS0637.cs  
using System;  
using System.Runtime.InteropServices;  
  
[StructLayout(LayoutKind.Explicit)]  
public class MainClass  
{  
   [FieldOffset(3)]   // CS0637  
   public static int i;  
   public static void Main ()  
   {  
   }  
}  
```
