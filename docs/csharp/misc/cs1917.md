# <a name="compiler-error-cs1917"></a><span data-ttu-id="93562-102">Erreur du compilateur CS1917</span><span class="sxs-lookup"><span data-stu-id="93562-102">Compiler Error CS1917</span></span>
<span data-ttu-id="93562-103">Les membres d’un champ en lecture seule 'name' de type 'struct name' ne peuvent pas être assignés à l’aide d’un initialiseur d’objet, car il s’agit d’un type valeur.</span><span class="sxs-lookup"><span data-stu-id="93562-103">Members of read-only field 'name' of type 'struct name' cannot be assigned with an object initializer because it is of a value type.</span></span>  
  
 <span data-ttu-id="93562-104">Les champs en lecture seule qui sont des types valeur peuvent uniquement être assignés dans un constructeur.</span><span class="sxs-lookup"><span data-stu-id="93562-104">Read-only fields that are value types can only be assigned in a constructor.</span></span>  
  
## <a name="to-correct-this-error"></a><span data-ttu-id="93562-105">Pour corriger cette erreur</span><span class="sxs-lookup"><span data-stu-id="93562-105">To correct this error</span></span>  
  
-   <span data-ttu-id="93562-106">Remplacez le struct par un type de classe.</span><span class="sxs-lookup"><span data-stu-id="93562-106">Change the struct to a class type.</span></span>  
  
-   <span data-ttu-id="93562-107">Initialisez le struct avec un constructeur.</span><span class="sxs-lookup"><span data-stu-id="93562-107">Initialize the struct with a constructor.</span></span>  
  
## <a name="example"></a><span data-ttu-id="93562-108">Exemple</span><span class="sxs-lookup"><span data-stu-id="93562-108">Example</span></span>  
 <span data-ttu-id="93562-109">Le code suivant génère l’erreur CS1917 :</span><span class="sxs-lookup"><span data-stu-id="93562-109">The following code generates CS1917:</span></span>  
  
```csharp  
// cs1917.cs  
class CS1917  
{  
    public struct TestStruct  
    {  
        public int i;  
    }  
    public class C  
    {  
        public readonly TestStruct str = new TestStruct();  
        public static int Main()  
        {  
            C c = new C { str = { i = 1 } }; // CS1917  
            return 0;  
        }  
    }  
}  
```
