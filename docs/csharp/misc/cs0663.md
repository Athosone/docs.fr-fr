 <span data-ttu-id="b0af8-104">Les méthodes qui diffèrent uniquement par leur utilisation de [dans](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) et [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) sur un paramètre ne sont pas autorisés.</span><span class="sxs-lookup"><span data-stu-id="b0af8-104">Methods that differ only on their use of [in](../../csharp/language-reference/keywords/in-parameter-modifier.md), [ref](../../csharp/language-reference/keywords/ref.md) and [out](../../csharp/language-reference/keywords/out-parameter-modifier.md) on a parameter are not allowed.</span></span>  
  
 <span data-ttu-id="b0af8-105">L’exemple suivant génère l’erreur CS0663 :</span><span class="sxs-lookup"><span data-stu-id="b0af8-105">The following sample generates CS0663:</span></span>  
  
```csharp

{  
   public static void Main()  
   {  
   }

   {  
   }  
}  
```
