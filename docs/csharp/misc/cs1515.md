# <a name="compiler-error-cs1515"></a>Erreur du compilateur CS1515
'in' attendu  
  
 Dans une instruction [foreach, in](../../csharp/language-reference/keywords/foreach-in.md) , la partie « in » est manquante.  
  
## <a name="example"></a>Exemple  
 L’exemple suivant génère l’erreur CS1515 :  
  
```csharp 
using System;  
  
class Driver  
{  
   static void Main()  
   {  
      int[] arr = new int[] {1, 2, 3};  
  
      // try the following line instead  
      // foreach (int x in arr)  
      foreach (int x arr)      // CS1515, "in" is missing  
      {  
         Console.WriteLine(x);  
      }  
   }  
}  
```
