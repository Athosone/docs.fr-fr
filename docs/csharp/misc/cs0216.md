# <a name="compiler-error-cs0216"></a><span data-ttu-id="a2ffe-102">Erreur du compilateur CS0216</span><span class="sxs-lookup"><span data-stu-id="a2ffe-102">Compiler Error CS0216</span></span>
<span data-ttu-id="a2ffe-103">L’opérateur 'opérateur' exige qu’un opérateur correspondant 'opérateur_manquant' soit aussi défini</span><span class="sxs-lookup"><span data-stu-id="a2ffe-103">The operator 'operator' requires a matching operator 'missing_operator' to also be defined</span></span>  
  
 <span data-ttu-id="a2ffe-104">Un opérateur [true](../language-reference/keywords/true-false-operators.md) défini par l’utilisateur exige un opérateur [false](../language-reference/keywords/true-false-operators.md) défini par l’utilisateur et vice versa.</span><span class="sxs-lookup"><span data-stu-id="a2ffe-104">A user-defined [true](../language-reference/keywords/true-false-operators.md) operator requires a user-defined [false](../language-reference/keywords/true-false-operators.md) operator, and vice versa.</span></span>
  
 <span data-ttu-id="a2ffe-105">L’exemple suivant génère l’erreur CS0216 :</span><span class="sxs-lookup"><span data-stu-id="a2ffe-105">The following sample generates CS0216:</span></span>  
  
```csharp  
// CS0216.cs  
class MyClass  
{  
   public static bool operator true (MyClass MyInt)   // CS0216  
   {  
      return true;  
   }  
  
   // to resolve, uncomment the following operator definition  
   /*  
   public static bool operator false (MyClass MyInt)  
   {  
      return true;  
   }  
   */  
  
   public static void Main()  
   {  
   }  
}  
```
